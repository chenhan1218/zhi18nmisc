摘要

本文件旨在說明如何使用 Linux 來設立一內部網路伺服器來與 Unix、Netware、NT 以及 Windows 連接在一起。因此，只要與 Linux 系統連接上， 你還能提供和所有其他平台透明的存取。 並提供了詳細解說如何使用NCSA 伺服器來設定 HTTP， 以及讓 Novell、 Microsoft 的 Windows3.1、WFWG、 Win95 及 WinNT 使用 TCP/IP 客戶端程式來連接上， 在 Apple 的PowerMac 上使用 MacTCP 來連接。
