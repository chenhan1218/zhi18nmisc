7. 其他事宜

7.1 Intelligent Agent

如果你需要 Oracle Intelligent Agent，我發現你不需修改任何設定便能執行它。要啟動 IA 的話︰

$ lsnrctl dbsnmp_start

要停止 IA︰

$ lsnrctl dbsnmp_stop

不會出現任何訊息說明啟動或停止 intelligent agent 是否成功。不過，在用戶端它有對 Enterprise Manager 作出回應，所以我假設它運作正常。
