5.5 Beowulf機器

  • Avalon 是由 140台Alpha 處理器組成，36GB記憶體，可能是最快的Beowulf機器，計算速度高達47.7Gflops，在全世界前五百快的機器中排名第114。
  • Megalon-A Massively PArallel CompuTer Resource (MPACTR)由14台個人電腦組成，每台電腦內有四顆Pentium Pro200處理器，總共有14GB記憶體
  • HIVE - Highly-parallel Integrated Virtual Environment 是另一套高速的Beowulf超級電腦，有64個計算節點，共計128顆處理器，4GB記憶體。
  • Topcat 是一套比較小型的機器，總共有16顆處理器和1.2GB記憶體。
  • MAGI cluster 是個有趣的網站，內有許多有趣的連結。
