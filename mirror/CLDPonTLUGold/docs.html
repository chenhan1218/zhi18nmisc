各種類型的文件

Linux 文件計畫(LDP)意在發行 Linux 作業系統下，對使用者有幫助以及可靠的文件。LDP 整個計畫的目的在於收集處理有關 Linux 問題的文件，範圍包括從線上文件(手冊頁、網頁…等)到包含安裝、使用與執行 Linux 的平面書。你可以連結下列兩個超連結得知 LDP 文件的聲明和權版授權書。

HOWTO 和 Mini HOWTO：

Linux 的 HOWTO 和 Mini HOWTO 是專為特定主題所寫的，也就是教你"如何"安裝、設定、處理某些問題。而 HOWO 索引則列出所有的 HOWTO 和 Mini HOWTO，並附上簡短的說明。可以到這裡，得知 HOWTO 和 Mini HOWTO 的最新更新消息。
