7. 我不需要準備什麼去使用 LBX？

遠端系統不需要一個新的 X server (如同一般的狀況，遠端系統並不需要任何一個 X Server).

你所需的程式並不需要指定特別版本的 X 或是特別的函式庫。 我借由 LBX 使用 X11R5 的程式，也沒碰到什麼問題。

你不需要使用 root 或是特別的權限存取選端系統。 你只需要一般的權限執行 lbxproxy。 另外你只需要在你自己的家目錄下執行即可，lbxproxy 並不需要安裝在特定的目錄。
