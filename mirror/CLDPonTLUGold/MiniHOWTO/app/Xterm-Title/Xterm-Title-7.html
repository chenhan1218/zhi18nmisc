7. 附錄: 其它語言的範例

寫一個小程式利用 xterm 的溢位功能把傳遞的參數當成主題. 以下是一些範例.

7.1 C 

#include <stdio.h>

int main (int argc, char *argv[]) {
  printf("%c]0;%s%c", '\033', argv[1], '\007');
  return(0);
}

7.2 Perl 

#!/usr/bin/perl
print "\033]0;@ARGV\007";
