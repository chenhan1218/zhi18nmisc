5. Windows 95 部分

在 win95 的 DOS 命令列裏建個開機磁片。這很重要哦，是在 DOS命令列裏做， 不是從 Windows 95內開個 DOS 視窗再做啊。拷貝 C:\WINDOWS95\COMMAND\SYS.COM 檔 到這張開機磁片裏 (如果你的路徑不同就改一下吧)。然後重新由這張磁片開機進入命 令列。執行 

SYS C:
這應該可以把 Windows NT OS loader 移除，就可以從 LILO 開機到 Windows 95 了。

現在你可以殺掉 NT loader 的幾個檔案: 

Boot.ini
Bootsect.dos
Ntdetect.com
Ntldr
