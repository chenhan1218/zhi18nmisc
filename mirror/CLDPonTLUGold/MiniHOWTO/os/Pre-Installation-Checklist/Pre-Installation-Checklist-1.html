這一份檢查表(aim)的幫助 ?

這是一份能夠讓你快樂安裝 Linux 的備忘錄.它在進一步的了解 Linux
中扮演了重要角色.
他將會幫助你準備安裝 Linux 再你的或是別人的電腦時所需要的答案.
有時候在安裝程序之中要找到一些細節是非常不方便甚至是不可能的.
當你在別人的辦公室裡安裝 Linux 到他的電腦,這些問題將會變得更困難.
當你填寫完這份檢查表,他將會幫助你在安裝之前找出某些硬體是否被你
計畫安裝的發行版本 (distribution) 支援.
他也將會幫助你準備你將要用到的套件 (packages) 並且做好必須的設定.
這是要減低你遇見令人苦惱的意外狀況.
這份檢查表是給一般個人電腦(PC)所使用的.你也可以將他用在其他的機種
上( 如 SUN, Alpha,...).



我能在哪裡找到更多有關於 Linux ?

如果你想要 Linux 專業 (PAID) 的幫助 -- [12]詢問 Algologic.
Algologic Research & Solutions 是被列在 LDP (Linux Documentation Project) 中官方文件裡的CONSULTANTS-HOWTO.

這一份檢查表並不會提供任何關於 Linux 的硬體相容細節,或是其他關於取得
Linux 或是安裝 Linux 的細節.
如果你想要知道這些細節, [13]請按這裡.
如果你是屬於沒耐心的人, 並且想要直接的安裝幫助
[14]請按這裡 並跳過安裝前檢查表
注意: 我們真的要被很多關於取得,安裝和使用 Linux 的一般問題淹沒了.
我們不會回答這一類的郵件.我們強烈的建議你 [15]點選這裡
取得其他的幫助資源. 請勿寄給我們任何一般性的 Linux 問題.

誰需要使用這份檢查表 ?

* 如果你計畫去安裝 Linux 在別人的電腦,或是在你工作地方之外的地方,
你一定預先要完成這一份檢查表
從你要安裝的人那邊取的這一份問的答案是很重要的,如此你才能夠對於
安裝 Linux 作適當的準備

* 如果你計畫要參加一個"安裝聚會" (我們印地安叫他做 "Linux mela"),
詢問跟你一起參加的人一起完成份檢查表的答案

* 相反的,如果你要去參加一個安裝聚會(也叫做 Linux mela)
去安裝 Linix 在你的電腦,作功課並回答檢查表上的問題 .

* 如果你要請某人過來幫你安裝 Linux ,寄給他/她這份問卷上你所知道的資訊

* 如果你是 D-I-Y 型的人,你必須開始回答這全部的問題,如此你會更節省時間
並享受你的 Linux!
