摘要

你是一個 Linux 新手嗎 ? 你是一個 Linux 老手嗎 ? 不論你是哪一者,你 會發現這一份檢查表對你有很大的幫助.因為常常有些必要的硬體資料我 們不知道,而有多少次在你摸索安裝過程中你能夠自己找到解決方法? 當你在安裝程序時是否曾經因為問題而感到困擾, 像是這樣簡單的問題: "這一個滑鼠是接到哪一個通訊埠(COM port)? 讓我們面對這問題,我們在一次或是數次安裝 linux 時會覺得我們應該在 開始安裝之前做更好的準備
