1. 簡介

這個 mini-HOWTO 是描述在 TCP/IP 網路上的 Linux system 如何 設定和別的系統共同分享裝在 Linux system 上的 MODEM. 譯者按：也就是在 TCP/IP 網路上如何把裝在 LINUX 上的一台 數據機分享給別的機器(如 Win95/NT)，讓這些機器以 為在自己這部機器上也裝上了一台數據機. 另外本篇文章只提到 Windows 上的設定方法，如果你 需要的是這以外的其它 OS 可能這篇文章就幫不上你 的忙了.
