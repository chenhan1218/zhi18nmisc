4. 安全問題

如果在你的區域網路中你只有一台 modem 來分享給所有機器，那麼就沒什麼安全上的問題可以顧慮了. 然而，如果你的區域網路上有一台或多台的機器是如上所述的被我們設定成 modem server 且也接上了 internet ，那安全上的問題就必須考量了. 否則任何人都可以用 "telnet your_host modem" 指令隨意撥一通國際長途電話.

我建議你為了要保護你的 modem server 被非法的存取你最好安裝以及設定 tcp-wrappers 來防止.
