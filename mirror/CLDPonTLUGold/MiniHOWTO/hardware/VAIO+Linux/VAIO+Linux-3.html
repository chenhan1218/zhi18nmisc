3. 如何安裝 Linux？

在這系統上安裝 Linux 不會有大問題，無論如何， 你 需要注意在一開始的時候，在系統之中有一個 non-DOS 的分割區 ，我不知道這個分割區是做什麼的，它是 400MB 大小的分割區， 可藉由執行 fdisk 看到，我直接把它刪除分配給 Linux 使用，而且我沒有什麼問題，所以它可能 是在軟體安裝時由某些程式產生的， (更新 12/29/98) 400MB分割區只在舊的型號上，實際上它應該是另外的 400MB DOS 分割區，但是 Sony 在這兒犯了錯誤，請依其它 一般的 Linux 安裝程序安裝， VAIO 有很好的支援 如果光碟片是可開機光碟片，你可以直接由光碟機開機， 因為 BIOS 支援光碟機開機，有一些 Linux 發行版本你可以使用這個方法，太棒了 因為你可以執行無磁碟片安裝，特別是， Redhat 可以從光碟開機，別的 Linux 發行版本像是 SUSE 也宣稱它 可以從光碟開機，但是我尚未確認這件事，我也不能確定這件事 ，如果你無法使用光碟開機的方法，試試 檢查 BIOS 之中的啟動選項，除了設定音效卡的部分 核心的設定應不是大問題，就像我先前提到的 ，在這段落我不會談到它， VAIO 電腦是配備有 APM BIOS，或者高階電源管理 BIOS，我 個人不建議你在 Linux 之下打開這個功能，當我打開這個功能時會有問題 ， (更新 12/29/98)我有收到關於 505 的報告 ，使用光碟機安裝很困難而網路的安裝是容易多了。
