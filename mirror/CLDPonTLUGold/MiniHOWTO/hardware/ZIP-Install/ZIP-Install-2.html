2. 介紹

本文分為四部分，每部分描述如何使用印表機版的 ZIP 機在一片 100MB 的 ZIP 磁片上安裝最基本的 Linux 系統。

第一部分描述如何設定 ZIP 磁片，適用於 Red Hat 和 Slackware 的版本上。 第二﹑第三﹑第四部分則分別描述如何在 ZIP 磁片上安裝 Red Hat 4.2, Slackware 2.2 和 Debian 1.2。

NOTE: 我了解到現在 Red Hat 5.0 已經發行，但在課業和工作中，(先讓我這 麼說吧)，在我有空在上面努力可能已經是五月底了。 而且我也希望能測試出 其他的版本。
