3. 在你開始之前...

3.1 你需要一個圖形介面的瀏覽器

這份文件假定你已經有了一個圖形介面的瀏覽器在系統上面執行, 例如 Netscape Navigator. 你必須有使用圖形介面瀏覽器的權限. 如果你要把Netscape Navigator用 在商業上,你可以向Caldera公司購買使用權.

3.2 你必須可以新增一個使用者帳號

如果你沒有成為root的權力, 請要求系統管理員新增一個 guest 帳號, 並把guest的home目錄的所有權交給你. 並跳到 ``新增或編輯以下的檔案'' 這個步驟 ( 在/home/guest中新增或編輯以下的檔案) 當他或者是她完成之後.

3.3 如果你要的是一台沒有網路連線, 但要讓大眾瀏覽你所提供的網頁內容的電腦, 你需要 httpd

如果你要的是一台沒有網路連線, 但要讓大眾瀏覽你所提供的網頁內容的電腦, 你必須有 httpd 在系統上執行, 以及你所要提供的文件. 如果你要知道你系統的情況, 輸入:

lynx -dump http://localhost/

你應該會看到你這台電腦上面所設定的html文件
