4. 新增 guest 帳號

以 root 執行 adduser 來新增一個叫做guest的帳號. 然後輸入

passwd guest

來設定guest的密碼. 最好將這個密碼設的簡單好記, 例如``guest''. 你將會告訴大家這個密碼. 所以別把他設定的和你自己的密碼一樣.

然後把 guest的home目錄的所有權轉移到你身上. 輸入

chown me.mygroup /home/guest

將``me''這個地方改成你的使用者名稱, 及把``mygroup''這邊 改成你的群組名稱. (在Red Hat Linux上, 也是一樣的作法, 就算每個使用者都有他自己的群組.)

你現在可以離開系統, 並將以下的步驟以你自己的帳號完成, 而不是root.?/p>
