6. 其它的 Quota 指令

6.1 Quotacheck

Quotacheck 是用來掃描檔案系統的磁碟用量，並更新 "quota.user" 此 quota 記錄檔到最新的狀態。我建議定期在系統啟動時執行 quotacheck 或藉由 cronjob 每隔一段時間執行一次（像是，一星期? ）。

6.2 Repquota

Repquota 產生檔案系統之摘要 quota 資訊。這裡有份 repquota 所得 的輸出範例:

# repquota -a 
                  Block limits             File limits 
 User       used  soft    hard  grace  used  soft  hard  grace 
 root  -- 175419     0       0        14679     0     0 
 bin   --  18000     0       0          735     0     0 
 uucp  --    729     0       0           23     0     0 
 man   --     57     0       0           10     0     0 
 user1 --  13046 15360   19200          806  1500  2250 
 user2 --   2838  5120    6400          377  1000  1500

6.3 Quotaon 以及 Quotaoff

Quotaon 是用來打開 quota 的計算； quotaoff 則是將其關閉。實際上 兩者很類似。它們是在系統啟動與關機時執行的。
