3. 軟體需求

留意事項 :

如果你的系統核心是 2.0 版以後的話,請跳到本文套件特有段落 (distribution specific section). 以下是針對 1.2版的核心而言.

這裡假設你已經安裝並可運作 Linux.

可由以下位置獲得 Token Ring 的修補程式:

ftp://ftp.wayne.esu1.k12.ne.us/pub/Linux/Token-Ring/TokenRing.patch-1.2.0.gz 可從下列位置獲得 Net-Tools 修補程式 ftp://ftp.wayne.esu1.k12.ne.us/pub/Linux/Token-Ring/net-tools-1.2.0.patched.tar.gz 替修補程式新增一個目錄 (例如 /usr/src/patches) 並把修補程式放在這. 

mkdir /usr/src/patches        central directory for patch storage
mkdir /usr/src/patches/token  place TokenRing patch here
