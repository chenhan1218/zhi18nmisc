1. 我的設定

  • 最新的核心 (2.0.27 - 從 ftp://ftp.funet.fi/pub/Linux/kernel/src/v2.0 取得) - 從 1.3.7x 以後就可以。
  • 將 IP Alias 編譯成可載入模組．你在用 "make config" 指令編譯核心時，如果要 把 IP Masq 編譯成模組(Modules)應會被問到．查看 Modules HOWTO(如果有的話) 或是查看 /usr/src/linux/Documentation/modules.txt 中的資訊。
  • 我在已分配給我的 IP 之外還又要了兩個額外的 IP。
  • 一片 D-Link DE620 網路卡(不重要，任何 Linux 支援的網路卡都行)。
