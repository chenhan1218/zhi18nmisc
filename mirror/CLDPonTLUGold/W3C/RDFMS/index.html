這份文件的狀態

這份文件已經由全球資訊網協會的成員及其它相關組織審查過了，並且已經由全球資訊網協會的主席背書而成為正式規格書，所以說這是一份穩定的文件並且可以當成參考資料或當成其它文件的正式參考。全球資訊網協會的角色是使得規格書專注於規範上並且促進規格書廣泛地部署。這會強化網路的功能以及網路跨平台的能力。

對於這份文件的建議可以寄到<www-rdf-comments@w3.org>。可在http://www.w3.org/Archives/Public/www-rdf-comments找到公開評論的文章。

RDF 稱為資源描述架構。
HTML 稱為超連結文字標記語言。
XML 稱為延伸標記語言。
PICS 稱為網路內容選擇平台。
MathML 稱為數學標記語言。
URI 稱為統一資源識別符號。
SGML 稱為標準通用標記語言。
