1. 介紹

自由軟體 在教育領域裡是一項很有價值的資源。它不僅在技術上和教學研究上比其他私有軟體優越，而且能進一步發揮 GNU 計畫在學校中的價值：

  • 自由
  • 合作

這裡是我們收集的有關 GNU 計畫和教育領域之間聯繫的一些資料（如果你想知道它們之間更多的聯繫，可以參見 GNU 計畫的歷史 )。請將你對這些資料的疑問和建議發送到 education@gnu.org。

2. GNU 中的教育領域軟體

除了 GNU/Linux， GNU 軟體 和 GNU 文檔, 還有一些專門為教育領域設計的 GNU 套裝軟體：

  • 喬治博士 是一個交互的圖形化工具，用戶可以通過它來探索和研究歐幾里德幾何學。
  • 成績簿 是一個老師用來掌握學生成績的圖形化應用程式。
  • GNU 打字員 是一個打字老師。它對打字的指導步驟進行解說，包含有幾種不同的語言和鍵盤布局。
  • Solfege 是一個輔助提高聽力的程式。
  • Gcompris 是一套適合年輕兒童的教育組件。

其它有用的有關教育方面的自由軟體可以在下列地址找到：

  • 由自由軟體基金會維持的 自由軟體目錄 中的 教育 這一部分。
  • OFSET 中的 freeduc 目錄。

如果你想了解更多的情況，或者你想加入一個有關教育方面的自由軟體的計畫，請參觀 Savannah Educational Projects，瀏覽一下已存在的計畫。或者請發郵件至 education@gnu.org 。

3. GNU 中的教育領域計畫

  • 自由的宇宙百科全書和學習資源
  • IT 報告 目錄的目的是提醒人們注意新的文章， 及建立一個關於自由軟體可以自由再分發的技術報告的書目。
  • OFSET 計畫是 GNU 計畫的一個合作計畫， 它主要開發屬於 GNU 套裝軟體的教育領域的程式。
