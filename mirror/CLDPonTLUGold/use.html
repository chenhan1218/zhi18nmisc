推廣 Linux 的使用

本計畫也是為了推廣 Linux 的使用而誕生的。由於 Linux 的固定成本(主要是軟體的成本)相對低廉，但變動成本卻高出許多(包含教育、訓練、維修、學習的成本)，所以總成本反而高出 Windows (由於還沒有實際的資料，所以無法做真正的分析)。

所以本計畫主要在於降低變動成本的支出而達到推廣 Linux 的使用(依照先後順序排列)：

  1. 宣傳本計畫
  2. 大量的中文文件
  3. 智慧型搜尋引擎
  4. 師資培育
  5. 校園出發

1. 宣傳本計畫

這相對比較簡單，由於總是會有新的 Linux 使用者，或許不知道本計畫，而需要花時間到處詢問問題。藉由宣傳本計畫，有問題的使用者可查看有沒有需要的中文文件，進而減少到處詢問所花的時間，使得變動成本降低。

2. 大量的中文文件

為了提供大量的中文文件，其中包含技術文件、使用者手冊、相對應的中文軟體與程式碼中文化，所以需要許多資金和人力。關於這個問題，則由基金會提供翻譯人員資金上的幫助，這樣可以使得翻譯有一定的量與質。那麼關於基金會的資金來源，簡單分類如下：

  1. 基金會將以財團法人的名義成立，所以募集的資金與所得的利潤不以營利為目的，而是為了推廣 Linux 的使用來運作。
  2. 從 Linux 社群募集資金，雖然這比較困難，但前人種樹，後人乘涼。所以本計畫還是會努力從 Linux 社群募集資金。
  3. 相關利益公司的贊助。也就是得到 Linux 相關公司的贊助與和微軟競爭的公司。
  4. 來自學校的資金，Linux 相關軟體的授權費較低，訓練成本偏高。資助本計畫，使得訓練成本降低，進而減少學校的軟體授權費。
  5. 線上雜誌，因為可以放入廣告。
  6. 政府資助，和校園提供資金是一樣的。
  7. 開發智慧型搜尋引擎，除了是殺手級的應用，而且也是本計畫必備的功能。

3.智慧型搜尋引擎

當中文文件一多之後，如何快速找到想要的文件就變成一大問題。就以買股票來說，分析師應該把所收集的資料整理分析好後，再告訴他的客戶。而現今的搜尋引擎就只是把一堆收集到的資料丟給自己，讓自己分析整理。所以本計畫先納入資源描述架構的功能，再推向語義網路的階段。也會和大學合作研究。

4. 師資培育

接下來就是要培養使用 Linux 的師資。先讓老師會用 Linux，那麼也才能在教學推廣 Linux 的使用。

5. 校園出發

最後就是要落實 Linux 的推廣，從校園出發--國小、國中、高中職、大學…等，依照學制的不同，把中文文件整理出適合學習的文件，並搭配影像、音效、互動功能。
