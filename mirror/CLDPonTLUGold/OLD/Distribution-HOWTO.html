本文件是預計用來幫助一個新的 user 如何選擇一種合適的 Linux 發行套件, 並且也幫助一個富有經驗的 user 如何掌握並瞭解 Linux 的市場現況. 本文件不想對所有電腦平台上運作的 Linux 發行套件作一份完整的列表, 相對的只是著眼於如何從 CD-ROM 光碟片上取得可在 Intel 平台上使用的主要英文語系(English-language) Linux 發行套件, 並藉此幫助初學者如何進入 Linux 之門.
