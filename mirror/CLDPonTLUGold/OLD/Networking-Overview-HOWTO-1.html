1. 介紹

本文主要在針對 Linux 作業系統的網路能力提出一個概觀的說明。 雖說使用 Linux 的好處是, 似乎各種領域都有豐富的資料來源, 但是大多偏重在實作方面。 對大多數的 Linux 新手而言, 特別是那些從 Windows 環境過來的人, 似乎難以消化吸收, 因為他們連 Linux 有什麼網路能力都還不知道。本文主要是向各位展示, 簡述每項網路能力的可能性, 並且給予各位更進一步的資料指引。 這些資料的收集來自許多方面: how-tos, faqs, 各計劃的網頁, 以及我個人的經驗。 我要將完成本文的功勞全部歸予以上的作者。沒有他們以及他們所撰寫的文件, 就沒有本文存在的可能或需要。
