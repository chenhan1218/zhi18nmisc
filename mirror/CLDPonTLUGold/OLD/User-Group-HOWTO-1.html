1. 簡介

1.1 主旨

這份文件旨在說明如何建立、經營 和發展 Linux 使用者協會。 Linux 是使用者可以自由拷貝的 Unix 系統， 在個人電腦、伺服器、乃至工作站上面都可以找得到它的蹤影。 Linux 是在 i386 的機器上研發出來的，而現在更支援 i486, Pentium, Pentium Pro, Pentium II 等微處理器， 以及 AMD, Cyrix 及其他廠商所生產的 x86 CPU。 此外它也支援各個版本的 SPARC、DEC Alpha、PowerPC/PowerMac、Motorola 68x0 和 Mac/Amiga 等機器.

1.2 如何取得進一步資訊

想多瞭解 Linux 嗎? 建議您從 Linux Documentation Project 開始。 （譯註: 中文版的 CLDP 可在 臺灣 Linux 使用者協會 找到）

至於一般的電腦使用者協會的相關資訊, 可以在 Association of PC Users Groups 找到。
