5. 何處取得 ?

你可從以下地方購買 Redhat Linux 光碟、Debian Linux 光碟或 Slackware Linux 光碟，從中找到 PostgreSQL 套件（原始碼和二元碼）︰

  • Linux System Labs 網址︰ http://www.lsl.com/ （7 美元）
  • CheapBytes Inc 網址︰ http://www.cheapbytes.com/ （7 美元）
  • Debian 主網址︰ http://www.debian.org/vendors.html

PostgreSQL 組織也售賣‘PostgreSQL CDROM’，內含完整的原始碼，許多 Unix 操作系統的可執行檔及完整的文件。

  • 從主網頁取得 PostgreSQL 光碟︰ http://www.postgresql.org 30（美元）

PostgreSQL 只含二元檔的發行版本 (distribution)︰

  • PostgreSQL RPMs 的維護人是 Lamer Owen，在 lamar.owen@wgcr.org 找到。
  • PostgreSQL 源 RPM 和二元 RPM http://www.ramifordistat.net/postgres。
  • PostgreSQL 源 RPM 和二元 RPM http://www.postgresql.org 。點選“Latest News”和 Redhat RPMs。
  • PostgreSQL 源 RPM 和二元 RPM http://www.redhat.com/pub/contrib/i386/ 。ftp 站台在 ftp://ftp.redhat.com/pub/contrib/i386/
  • Solaris, HPUX, AIX, IRIX, Linux 的二元檔站台︰ ftp://ftp.postgresql.org/pub/bindist
