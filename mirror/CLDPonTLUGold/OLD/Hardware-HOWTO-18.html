18. 滑鼠

18.1 有支援的

  • Microsoft serial mouse
  • Mouse Systems serial mouse
  • Logitech Mouseman serial mouse
  • Logitech serial mouse
  • ATI XL Inport busmouse
  • C&T 82C710 (QuickPort) (Toshiba, TI Travelmate)
  • Microsoft busmouse
  • Logitech busmouse
  • PS/2 (auxiliary device) mouse

18.2 其他

  • Sejin J-mouse
    ftp://sunsite.unc.edu/pub/Linux/kernel/patches/console/jmouse.1.1.70-jmouse.tar.gz
  • MultiMouse - 使用多個滑鼠設備像只有一個滑鼠一樣
    ftp://sunsite.unc.edu/pub/Linux/system/misc/MultiMouse-1.0.tgz
  • Microsoft Intellimouse

18.3 註

觸控板Touchpad 設備，像 Alps Glidepoint 也可以使用，雖然他們相容於另一種滑鼠協定。

新型的 Logitech 滑鼠 (除了 Mouseman之外) 採用 Microsoft 協定，三個按鈕都可以使用。 雖然 Microsoft 的滑鼠只有兩個按鈕，這個協定允許使用三個按鈕。

ATI Graphics Ultra 和 Ultra Pro 的滑鼠部份是採用 Logitech busmouse 協定。 (詳情請看 Busmouse HOWTO)
