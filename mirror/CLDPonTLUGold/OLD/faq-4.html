4. 其它

4.1 除了翻譯，我還能做什麼?

答: 您還可做這些:

  • 協助建立翻譯常用辭彙。請參考 翻譯小辭彙 的說明。
  • 幫忙找出 CLDP 文件的打字，排版等錯誤。
  • 提供 FTP 機器做為本中譯計畫的正式站台。 如果您願意的話，請注意您的機器最好是跑 Linux，而且必須夠新，夠快，夠穩(不能常 crash)，最重要的是，必須要有足夠的網路頻寬! 還有這機器必須能長久存在。
  • 提供機器 Mirror 這些中譯文件。Mirror 的站台就沒有上面那些要求了，只要您願意就好!
  • 為本網頁設計 LOGO, 加上一些美麗的圖片吧!
  • 任何的建議，批評，點子，or WHATSOEVER。
  • 提供金錢贊助，數目隨意……

4.2 我想翻譯/寫作其它與 Linux 相關的文件，可以放在 CLDP 網頁上嗎?

答: 是的! 相當歡迎。將它寄給我即可。不過因為要放上網頁，請以 HTML 或其它易於轉換為 HTML 的格式寫作。

4.3 除了翻譯 Linux HOWTOs, CLDP 可還有其它的計劃?

答: CLDP 目前還沒有其它具體的計劃。但相當歡迎有人能提出新計劃的想法與建議。 例如翻譯線上手冊(man pages), 翻譯 Linux Journal, 中文 Linux 每日新聞等都是不錯的點子。

一個計劃必須有明確的方向與詳細的執行步驟才能有所成。因此如果您有好的想法，不妨來信和我討論。 只要能提出具體可行的計劃，相信必能引吸許多 Linux 高手的參與。

4.4 你是誰?

答: 我是 CLDP 計劃的發起人，也是目前計劃的協調人。我負責接收各位譯者寄來的文件，轉換後發表在網頁及其它的地方。 我也維護了 HOWTO INDEX 的中譯版，以及 Linux Chinese HOWTO 等文件。

我也對任何 Linux 中文化或推廣 Linux 使用的活動有興趣。如果您有這些想法或建議，歡迎來信。
