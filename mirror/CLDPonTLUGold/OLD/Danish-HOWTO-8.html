8. 資訊資源

8.1 其他相關文件

你可以從 www.linuxdoc.org 的任一映射站取得這些 HOWTO. SunSite DK 是一個丹麥映射站.

Winfried Trümper 寫的德語 HOWTO. 其他像是芬蘭語, 西班牙語, 和波蘭語都已經有了以母語寫成的 HOWTO.

Andries Brouwer 寫的 Linux 鍵盤和控制台 HOWTO.

這裡有 ISO 8859-1 National Character Set FAQ, Programming for Internationalization, 以及許多其他 Michael Gschwind 寫的文件.

8.2 FTP 及網站

SSLUG (Skåne Sjælland Linux User Group) 是一個瑞典語/丹麥語 Linux user group. 你可以從他們的信件群組中得到許多以丹麥語(或瑞典語)寫下的Linux相關問題. 這個 user group 同時負責維護 這份文件.

Ålborg 的 AUC 的主站是 SunSite DK. 它們發行 Debian 及 Red Hat 版本的 Linux, 最新的 kernel, 以及 Linux Documentation Project, metalab.unc.edu 及 the GNU archive 的映射站. 另外, 你可以從 the CTAN archive 這個映射站得到一切和 TeX 與 LaTeX 相關的資訊.
