5. 本地 LUG 的活動

上節主要討論 LUG 能進行何種工作和應該進行何種工作。本節集中討論以何種方式完成這些工作。

儘管形式不同，但地區性的 LUG 可以進行兩種工作：第一種是會員在實際的會議場所舉行會議； 第二種是他們通過網際網路相互聯系。幾乎所有 LUG 能夠進行的工作均可以會議和網上資源表示。

5.1 會議

如我前文所述，舉行實際會議簡直就是 LUG 和多數電腦使用者協會的所有工作。LUG 可以舉行下列各種會議：

  • 交誼會
  • 技術發表會
  • 非正式討論小組
  • 使用者協會的一般事務會議
  • Linux安裝示範
  • 設定配置和糾錯討論

LUG 在這些會議上做什麼？

  • 為新手安裝 Linux
  • 教導會員 Linux 的用法
  • 比較 Linux 和其他操作系統的優劣
  • 教導會員在 Linux 上運行的軟件
  • 討論推廣 Linux 的方法
  • 討論自由軟件運動的重要性
  • 討論使用者協會的一般事務
  • 吃喝玩樂，共享快樂時光

5.2 網上資源

網際網路在商業上的崛起與 Linux 的受寵幾乎同時發生，而後者多少是由于前者而四處流行。 網際網路一向都是 Linux 發展的重要伙伴。這對 LUG 而言並無不同。多數 LUG 都有網頁，有的 LUG 甚至擁有整個網址。事實上，如果不到網上查找，還不知道怎樣找到一個地區性的 LUG 呢。

因此，一個地區性的 LUG 充分利用能夠掌握的網際網路技術是完全合情合理的事，不論是使用網址、 電子郵件、WAIS、finger或新聞郵區等。正如商家發現的那樣，網際網路也可作為推廣、通知、教導 和甚至推銷的有效工具。LUG廣泛利用網際網路技術的另一個原因正是 Linux 為這種 技術的使用提供了一個非常穩定的環境。因此不僅 LUG 得益于設立網址，LUG 可以利用 網址宣告它的存在並有助于組織會員之間的活動，同時在發展這種技術時，LUG 會員還有機會 學習這項技術，並親身體會 Linux 的運作。
