正文:
沒有一種只以單一方式發行散布的 Linux 發行套件. 換句話說, 有很多像這樣既可經由匿名的 FTP 站台下載也可藉由郵購光碟的方式來取得的發行套件.

這份文件的目地是用來對以光碟方式存在的英文語系 Linux 發行套件提供一個簡短的摘要說明, 並且也提供讀者如何尋找更多資訊的指引. 在此之前有一份由 Marco Budde 維護的德文 Distribution HOWTO, 但現在似乎已經消失不見了. 我們不知道除了英文及德文語系之外還有其它語系的發行套件公諸於世.

這裡所呈現出來的資訊不能說是很完整; 因為除了這裡所列舉出來的以外還有其它的 Linux 套件. 雖說基於開放性原始碼 (open source software) 的本質, 任何人可在他們的安裝過程中作任何的改變並以其它方式架構它使其成為 `另一種發行套件(a distribution)'. 如果一份文件像這樣時必須找個地方加以標示出來. 我也標示它為 `available on CD'. 但這絕不是用來作為 Linux 發行套件的定義的. 有為數眾多的小型發行套件可提供一些東東但大型的發行套件們就不會這樣做, 因此有不少它本身是毫無可取之處的.

這裡所呈現出來的資訊不能說是很完整; 除了這裡所列舉出來的以外還有其它的 Linux 套件. 假如您有聯想到某種 CD-ROM 版本的發行套件且是這裡所沒有的, 請看接近 對本文件的提案 這節的尾端以獲得提出提案時相關的資訊. 它是簡單易行且將用不到五分鐘就可完成的.

1.3 Linux 市場概觀

在早期的年代(大概是西元 1993 年吧), 您從網路下載一個 Linux 發行套件到軟碟片上是一件很重大的事. 由於當時沒有很方便的儲存媒體導致在整個安裝的過程中變得很吃力及失敗的反覆發生都是很常見的.

後來 CD-ROM 光碟機及 CD 光碟片變得越來越便宜, 一種適合傳送大量軟體及作業系統的理想儲存媒體變得更為物美價廉. 一個健全小型企業模式的軟體工業正在商業 Linux CD-ROM 光碟市場中建造了起來, 他們並且逐漸地主導 Linux 的舞台(因為這些販售者有實質的金錢可以作為市場行銷及服務支援的資金). Debian 是現在唯一僅有深富意義的非商業性版本的 Linux 發行套件, 甚至它透過 shovelware 光碟似乎更加廣泛地傳播推廣開來了.

現在絕大部份以光碟型式發行的套件(包括 Slackware, Yggdrasil and Red Hat) 仍然可從這些公司發展部門的自家站台以 FTP 的方式取得相同的發行套件. 但假如您有光碟讀取裝置且經濟能力也不錯的話, 您將對這些套件及技術支援有更大的選擇空間(並且您也可經常性的取得一些有用的資訊文件). 您可從這個網址
http://metalab.unc.edu/mdw/HOWTO/Installation-HOWTO.html 尋找這份文件 the Linux Installation HOWTO 以得到更多有關安裝的詳細說明.

可在 Intel 平台上執行的 Linux 光碟版發行套件其定價約從 $20 美元起跳, 而其最高的總額約為 $50 美元左右 (另加少許費用您可買到更具價值的東東). 很多的販售者有提供預約的交易服務, 它能在預約的期間內作定期的更新以降低您的 CD 單位成本( cost-per-CD ).

功能及品質幾乎可和價格多寡關聯在一起(所以像在一個非常競爭市場下的產品將是眾人所期待的). 作者建議您多支付少許的金錢來購買最重要的(top-drawer)原版 CD 光碟套件; 這樣您在解決安裝及管理這些麻煩的事才會變得比較輕鬆愉快.

現在作好一個選擇是比以前還要簡單得多了. 在 1995-1996 年間 Linux 市場經過一場嚴厲的淘汰期, 當那些體質不佳的競爭者消失或一厥不振時有一些商業版發行者展露頭角成為領導者. 而在一般用途非商業版的發行套件之中其淘汰情形更加的激烈, 基本上來說, 只有 Debian 在這場競爭之中生存下來.

就整個結論來說, 以前曾經規範了市場輪廓的三階組織結構 (three-tier structure) 的原始套件建造者(primary distribution builders), 附加價值 (value-added) 的再包裝者及低價供應(bottom-feeding) CD 光碟片的 shoveller 最近幾乎已經一厥不振了. 如果想在 1997 年使自己具有競爭力, 那就要在 Linux 的配備中(無論是否為商業版或是非商業版), 必須要像一個主流的套件建立者提供合理範圍內的支援和表現, 不論它是否真的可行. 所以只要您期待一個最近的版本發表日期, 那麼目前也不會因不得不使用一個不佳的套件而感到相當的難受.

1.4 筆者的忠告

上一章節中所提到的都是一些事實及真相. 而在本章節中要講的是我的看法(不管我的意見是否有價值 -- 請記得免費忠告的警語). 根據您對這份指導書所得到的經驗與資料而做出您自己的評估將是無可取代的, 在這裡我將盡我所能地說明我可能的偏見而不是為文告訴您應該要怎麼做.

在從 1993 年到 1995 年的秋天, Yggdrasil 一開始就是 Linux CD 業中的行中翹楚 -- 基本上來說, 它開創了光碟版市場規模外也在當時為大家豎立了標準. 我習慣於使用 Yggdrasil, 並且我建議它是因為它大量的優質文件超越了商用的 System V 版本, 其廣泛收集的應用軟體, 和寄發免費版本給自由軟體寫作者及奉獻出每套光碟軟體部份的金錢作為自由軟體發展財政上支援的開明策略. 但是 Yggdrasil 自從 1995 年後就沒有再發行新的版本並被市場所遺忘了.

我現在使用 Red Hat Linux 系統並滿意極了. 現在 Red Hat 公司的 RPM 技術正賦予它, IMO, 此科技能力的鋒芒超越了其它的販售者. 他們在適當的時間裡已完成了大部份合宜的行動並且我相信這些皆使它成為現有市場的領導者.

假如您執意地要使用一個非商業版本的發行套件, Debian 對我來說似乎是一個明智的選擇, 但您要注意它的技術支援服務會是一個嚴重的問題.

上面這些觀點當然不能解釋成為一份無條件的保證; 不同 Linux 的發行套件是可針對不同的須要作最佳化的, 並且您的系統可藉由一些其它的套件調整成最佳化(我的情形和您們不一樣, 特別是您主要是使用 DOS 並正尋找一個可調整為雙重開機並由 DOS 啟動的發行套件).

再者, 工業標準是極易生變的. 在您閱讀本文的同時, Red Hat 或 Debian 的競爭力可能已經減弱並已由更為積極, 企圖心旺盛的新競爭者加以取代了.
