這份文件是一份 "實際操作的說明" ，以便於能使您很快的設定 CVS/RCS 原始程式碼控制系統。這份文件裡也有一些將 CVS 上常用的混合命令包成可設定的 shell scripts 。這些 scripts 為 CVS 提供了一個簡單的使用者介面。這份文件的內容不僅能適用於 Linux 系統也適用於其他像 Unix 的系統，例如Solaris, HPUX, AIX, SCO, Sinix, BSD, SCO 等等。
