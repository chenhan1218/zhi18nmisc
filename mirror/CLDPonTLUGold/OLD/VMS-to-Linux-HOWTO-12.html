12. 不可少的小偏方

  • Command completion: 當您在命令列欲輸入某一指令的時候﹐您不需要完全輸入這個指令的所有名稱﹐ 輸入前幾個字元後﹐再按 <TAB> ﹐系統會幫您完成輸入所有的指令名稱; 例如: 您必須輸入 less this_is_a_long_name; 現在您只要輸入 less thi<TAB> 系統就會幫您輸入剩下的部份﹐不需要再書如全部的檔名了。 (如果您有很多檔案都是以這些字元作開頭﹐當然系統就會顯示出所有以這幾個字元開頭的檔案﹐但是﹐您可以輸入足夠的字元﹐就可以得到您需要的唯一檔案。)
  • Back-scrolling: 同時按 SHIFT--PAG UP 可以讓您的螢幕往回捲到以前顯示過的﹐ 但是可以往回捲幾頁﹐則需要視您的顯示卡的記憶體而定了。
  • Resetting the screen: 不果您不小心用 more 或 cat 開啟了一個二進位檔(binary file), 那您的螢幕可能為因此而亂掉。這時候您可以使用 reset 或是使用 echo CTRL-V ESC c RETURN 可以讓您的螢幕恢復正常。
  • Pasting text: 在 X 下﹐以滑鼠在 xterm 視窗拖曳出一個區域﹐ 然後按滑鼠中鍵（如果您的滑鼠沒有中鍵﹐可以同時按滑鼠的左右鍵模擬中鍵）就可以貼上您複製的文字。
  • Using the mouse: 在終端機模式下﹐您可能需要尋為您的系統管理員有沒有安裝 gpm 這隻程式, 如果有﹐您就可以用滑鼠拖曳出一個區域﹐然後按滑鼠的左鍵﹐就可以貼上您剛剛拖曳出來的文字區域。 （不同的虛擬終端機(VC)可能會有不一樣的操作方式）。
