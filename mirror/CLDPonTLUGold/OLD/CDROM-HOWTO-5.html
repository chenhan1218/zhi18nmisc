5. 應用

這一節簡要地列出在 Linux 下 CD-ROM 的主要應用。 查閱 Linux軟體大地圖(Linux Software Map) 看看它們在何處及最新是第幾版。

5.1 CD唱片播放程式

有好些程式有能力放CD 唱片，可從耳機洞或音效卡出聲。

Workman

一個在 X11 下的圖型化播放程式，支援 CD資料庫，還有很多其它功能。

WorkBone

一個互動性的文字模式播放程式

xcdplayer

一個 X11 下的簡單播放軟體

cdplayer

非常簡單的命令列播放器

Xmcd

一個 X11/Motif 的播放軟體

xmitsumi

另一個給 Mitsumi 機器用的 X11 下播放程式

xplaycd

另一個 X11 的播放器，包含一個混音器及 VU 計程式

cdtool

命令列播放 CD 唱片的工具

以上有些程式可能程式裡規定用某個設備檔(如 /dev/cdrom)。 也許您可以用參數指示它用什麼設備檔，或您也可在 /dev 下建符號連結來指示。 如果將聲音送到音效卡，您或許會想用混音器來調音量或選擇 CD-ROM 輸入當錄音音源。

5.2 相片光碟

相片光碟利用 ISO-9660 檔案系統存放專屬格式的影像檔。 並非所有 CD-ROM 都支援讀取相片光碟。

Hadmut Danisch 寫的 hpcdtoppm 程式可將像片光碟檔案轉成可轉移式點陣圖 (ppm，portable pixmap) 格式。 它可經由 ftp://ftp.gwdg.de/pub/linux/hpcdtoppm 取得，或者也可以在 PBM(portable bit map)公用程式(在很多檔案庫中都都可取得) 裡 面找到。 (請查詢 ``pbm'' 或 ``netpbm'')

Gerd Knorr( kraxel@cs.tu-berlin.de)寫的 photocd 程式可將相片光碟轉成 Targa 或者 Windows 及 OS/2 用的 BMP 檔。

這位作者也寫了 xpcd，一個可處理相片光碟影像的 X11 軟體。 您可以用滑鼠選擇影像， 在小視窗中瀏覽，或用五種解析度看它。 您也可選定影像的一部份，只載入它們。 xpcd 可 在 ftp://ftp.cs.tu-berlin.de/pub/linux/Local/misc/中找到。

ImageMagick 影像檔操作程式也支援像片光碟檔案。 它在 ftp://ftp.x.org/contrib/applications/ImageMagick/裡。

5.3 Mkisofs

Eric Youngdale 的 mkisofs 套件允許您在硬碟上建立 ISO-9660 檔案系統。 這樣可以幫助您深入了解光碟，也能在燒光碟前先測試其檔案系統。

真正將資料寫進CD 燒錄機的工具幾乎都依廠商不同而有所不同。 它們也要求 寫資料時不能有任何中斷，所以像 Linux 這種多工作業系統並不是非常適合這種工作。

5.4 ISO-9660 公用程式

有些公用程式可以確認 ISO-9660 格式化的碟片是否正確。 您也許會發現在測試奇怪的 CD 時會有用。 這玩意可在 ftp://ftp.cdrom.com/pub/cdrom/ptf/ 中找到。 作者是 Bill Siegmund 及 Rich Morin。
