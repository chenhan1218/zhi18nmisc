15. CD-ROM 磁碟機

關於 CD-ROM 磁碟機的進一步資訊，請參閱 CDROM-HOWTO http://sunsite.unc.edu/LDP/HOWTO/.

15.1 有支援的

一般的 CD-ROM 磁碟機

  • SCSI CD-ROM 磁碟機
    (來自 CD-ROM HOWTO) 任何 SCSI CD-ROM 磁碟機不管是 512 或 2048 bytes 的區塊都可以在 Linux上使用； 這包含了在市場上佔大多數的 CD-ROM 磁碟機。
  • EIDE (ATAPI) CD-ROM 磁碟機 (IDECD)
    幾乎所有的雙倍、四倍和六倍速都被支援，包括
    • Mitsumi FX400
    • Nec-260
    • Sony 55E

15.3 註

所有的 CD-ROM 磁碟機在讀取資料上應該都是一樣的。在音樂 CD 播放工具埵陶\多的相容性問題。 (特別是新型的低階 NEC 磁碟機上)有些測試版驅動程式也許還沒支援音效。

早期的 (單倍速) NEC CD-ROM 磁碟機在現行的 SCSI 控制器上也有問題。

PhotoCD (XA) 有被支援。由 Hadmut Danisch 所寫的 hpcdtoppm 程式可以將 PhotoCD 檔案轉換成為可攜式的 pixmap 格式。 你可以從 ftp://ftp.gwdg.de/pub/linux/hpcdtoppm 這堥得該程式，它也是 PBM 公用程式的一部份。

另外，讀取 video CD 已在 kernel 2.1.3x (含)以後的版本堣銧岸F。給 kernel 2.0.30 的修補檔也出來了。

最後，大部份的 IDE CD-ROM Changers 也被支援喔。
