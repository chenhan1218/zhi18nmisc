1. 什麼是 Linux HOWTOs?

Linux HOWTOs 是一份詳細描述在某一方面規劃或使用 Linux 的文件。 例如，有一份 Installation HOWTO，說明如何安裝 Linux，而 Mail HOWTO 說明了如何在 Linux 下建立及設定電子郵遞系統。 其它還有 NET-3 HOWTO 及 Printing HOWTO 等等。

HOWTOs 是易於理解的文件 - 和所謂的 FAQ 很像，然而並非使用問答的形式。 不過許多的 HOWTOs 在結尾的地方包含了 FAQ 的部份。

HOWTO 有許多的文件格式: 純文字, PostScript, DVI 以及 HTML。

除了這些 HOWTOs，還有許多的 mini-HOWTOs。它們是較短的，特殊的 HOWTOs。它們只有純文字及 HTML 的格式。
