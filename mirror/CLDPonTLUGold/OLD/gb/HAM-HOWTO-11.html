11.1 oscope - Enhanced Software Oscillosope -- 增强型软件示波器

作者

Tim Witham, twitham@pcocd2.intel.com

简单说明

`oscope'是一个 44 KHZ 的软件示波器，它使用的设备是 /dev/dsp。它基于 Jeff Tranter 的源程序，并做了很优良的性能改进。

当前开发状态

首次产品发布

更新日期

1997年4月1日

系统需求

需要支持 8800Hz - 44kHz 声卡的Linux 内核，基于 X11 的 xoscope 需要`libsx' 包，基于控制台的 oscope 需要 `svgalib' ，`g3vga' 用来向控制台添加字符，其它的程序用来选择输入及决定输入电平。

详细说明

本软件包含支持 SVGA 的 Oscope 及 支持 X11 的 Xoscope。44KHZ 2 通道 8 比特取样，5 us/div 到 5 ms/div 的可变时间轴，5 个自动量程，内部或外置的数学处理功能如求和，差，平均值，均方根等，26 个记忆频道，同时可显示 8 个输入，文件的保存/载入。

从何处可取得本软件。

你可以从下面的站点： www.bobsplace.com 或 sunsite.unc.edu 取得 oscope 。

使用许可/版权

GNU 公共许可, 自由软件, 无品质保证。

软件提供者：

Tim Witham

11.5 irsim -- MOS 逻辑级模拟软件

作者

Dmitry Teytelman, dim@leland.stanford.edu

简单说明

一个事件驱动的逻辑级 MOS 电路模拟器。

当前开发状态

版本：9.2,成品。

更新日期

1997年4月1日

系统需求

X-Windows.

详细说明

irsim 是一个基于 X11 的 MOS 电路模拟器。它有两种模拟方式。switch 方式把三极管模拟成压控开关，而 linear 方式把三极管模拟成电压控制的连续可变电阻，每种模式都有一个寄生电容。

Spice 允许你在不用接触电烙铁和焊锡丝的情况下通过计算机的模拟环境来设计、模拟，测试电路的运作情况。软件包里有 readme 文件指导你的使用。我是按照下面的步骤安装的：

# ./utils/build linux install
