v6.3, 8 May 1999. 翻译完成日期: 12 Aug 1999

本文件是预计用来帮助一个新的 user 如何选择一种合适的 Linux 发行套件, 并且也帮助一个富有经验的 user 如何掌握并了解 Linux 的市场现况. 本文件不想对所有电脑平台上运作的 Linux 发行套件作一份完整的列表, 相对的只是著眼於如何从 CD-ROM 光碟片上取得可在 Intel 平台上使用的主要英文语系(English-language) Linux 发行套件, 并藉此帮助初学者如何进入 Linux 之门.
