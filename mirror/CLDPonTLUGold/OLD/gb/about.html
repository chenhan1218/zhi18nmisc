关於 CLDP 计划

拜网际网路风行之赐，许多资讯得以快速流通，也使得许多梦想得以实现。Linux 就是其中一例。

Linux 是一套免费的作业系统。所有的原始程式码完全公开。 任何人皆可自由取得，散布，或是修改其程式。Linux 是真正 32 位元，多人多工的作业系统。 它的稳定性，多工能力与网路功能，是许多商业作业系统根本无法相比的。 (关於 Linux 的介绍请参阅 Linux META-FAQ 及 INFO-SHEET 的进一步说明。)

但不可否认的，Linux 并不是的一套容易上手的作业系统。初学者往往需要大量的时间摸索，熟悉，才能够驾御自如。文件的缺乏恐怕是最大的原因。 事实上今日关於 Linux 的文件与书籍已经不少，不过真正经过有条理系统整理的却不太多，且多半是英文文件。 中文的文件是少之又少。

Linux 中文文件计划(Chinese Linux Documentation Project, CLDP), 目的正是要为 Linux 提供丰富，完整，高品质的中文文件。

Linux HOWTOs 是一个整理完善，更新快速的 Linux 文件收集。它是初学者最佳的入门文件。 因此我们选择了 Linux HOWTOs 的翻译计划做为我们的开始。

如果你也是 Linux 的爱用者，有一颗回馈服务的热心，请加入我们的行列，为 Linux 的中文化尽一份心力!
