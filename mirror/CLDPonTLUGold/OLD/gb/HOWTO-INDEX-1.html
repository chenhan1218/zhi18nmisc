1. 什么是 Linux HOWTOs?

Linux HOWTOs 是一份详细描述在某一方面规划或使用 Linux 的文件。 例如，有一份 Installation HOWTO，说明如何安装 Linux，而 Mail HOWTO 说明了如何在 Linux 下建立及设定电子邮递系统。 其它还有 NET-3 HOWTO 及 Printing HOWTO 等等。

HOWTOs 是易于理解的文件 - 和所谓的 FAQ 很像，然而并非使用问答的形式。 不过许多的 HOWTOs 在结尾的地方包含了 FAQ 的部份。

HOWTO 有许多的文件格式: 纯文字, PostScript, DVI 以及 HTML。

除了这些 HOWTOs，还有许多的 mini-HOWTOs。它们是较短的，特殊的 HOWTOs。它们只有纯文字及 HTML 的格式。
