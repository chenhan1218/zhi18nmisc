4. TeXing

TeX 和 LaTeX 在 Linux 上完全和 VMS 一样---不同的地方是，Linux的执行速度更快 :-), 这个工具用来处理 .dvi 和 .ps 的档案更俱优势：

  • 由 TeX 执行 TeX 档，就像平常一样： tex file.tex;
  • 转换 .dvi 挡到 .ps 档，输入 dvips -o filename.ps filename.dvi;
  • 检视 .dvi 档，在 X Window 下输入： xdvi filename.dvi &. 在显示页按下滑鼠可以放大。 这是一个智慧型的程式：如果您编辑且执行 TeX 产生一个较新版本的 .dvi 档案，xdvi 会自动更新;
  • 检视 .ps 档， 在 X Window 下输入 ghostview filename.ps &. 按滑鼠可以缩放。 全部的档案或是部份档案可以被列印出来。 然而还有一个比 TeX 更好的程式：gv.
  • 列印 .ps: 通常指令是 lpr mypaper.ps ，但是如果 PostScript 列表机被呼叫（假设使用 ps 指令） 您将会是执行： lpr -Pps mypaper.ps。对於更多有关 print queues 的讯息，请参照 列印埠.
