Linux 网路资源

与 Linux 有关的网站已不胜枚举。这里仅列出我们认为最重要，最有用的。 如果您发现值得推荐的站台，别忘了告诉我。

更新日期: 1999/12/26 15:15

中文站台:

BIG5 码

Linux @ Taiwan
台湾 Linux 使用者组织。也是 CLDP 的主站台。
中文 Linux 延伸安装套件
架构在 Red Hat Linux 上的中文延伸安装套件。 终极目标在架构一个完全中文化的安装套件 (distribution). 极力推荐。
LinuxFab
Linux 及时新闻。立志成为台湾的 Slashdot...:) 目前正对 Linux 的商业应用或研究计画进行调查。如果您正在使用 Linux 可至该站登录。
LinuxFocus 线上杂志中译版
LinuxFocus 电子杂志中文翻译。继 CLDP 後另一个重要的 Linux 文件翻译计划。急需大家多多参与。
朝阳资管洪朝贵副教授的首页
有许多介绍自由软体，宣扬 GNU 理念的好文章。
XCIN Project
XCIN 专案计划(X Window 的中文输入法)的网页。
Linux Notes
站长用 Linux 写日记，记录下测试过的各种好用软体。还有站长自己写的程式哦。很不错的一个个人化网站。
GNU Translation Project - Chinese Team
一个将 GNU Tools 讯息中文化的翻译计划。需要大家多多参与。
Unix-like OS for PC
以介绍 Red Hat Linux 为主的站台。内容丰富，Red Hat 的使用者不可错过。站长 marr.
Linux SAGA (原 OkStation)
站长 foxman 撰写及翻译了不少关於 Linux 的好文章。 包括很多 Linux 及 Windows NT 的比较。同时提供 FTP, CLDP 映射站等。是个相当用心经营的 Linux 站台。
微软万圣节备忘录事件
微软万圣节备忘录是由微软内部所外□的文件, 造成媒体及业界相当程度的震憾。 本文件对开放性原始码的成功原因，以及对微软帝国造成的威胁有深入的剖析。 OkStation 站长大手笔将此事件一系列相关文件中译，值得大家鼓掌赞扬。
台大电机 Maxwell BBS 精华区
一个整理得相当完善的 Linux 精华区。
中研院 ASPAC 计画文件
这是中研院内部的研究计画，将多种公用软体分类、测试後，整合。有许多好用工具的中文说明。
台湾 Linux 资源网
尝试要从企业观点、商业观点的角度，做 Linux 相关资源的收集与推广。 同时提供民调、双向讨论区等等互动服务，来收集台湾本地 Linux 使用族群的状况。
探索 Linux
定期推出 Linux 应用的相关文章。同时有线上讨论区。
交大资科 Linux 站
现有 FTP , World Wide Web, NFS-Install 等服务。 致力於本土 LINUX 作业系统的推广与应用。 站上有不少中文文件，软体套件等。并映射许多著名的 Linux 站台。
光碟月刊 Linux 文章索引
收集了曾经刊在光碟月刊(现改为 Internet Pioneer)上的 Linux 文章。
Clem's XEmacs Page
说明如何在 XEmacs 中设定使用中文。
Hanky's Linux Site
有 Linux Kernel 2.2.x 的测试报告，以及不少特殊的主题。
HKLUG Linux News
由香港 Linux 使用者协会所维护的 Linux 中文及时新闻。可惜目前计划已停止。
Unix/Linux - voidspace
提供相当丰富的 Linux 中文资讯。站长就是上面提到的 Linux 中文新闻的维护者。
百资科技
代理各种 Linux 相关产品，及数十种程式语言工具。还有 Linux 的讨论区。

GB 码

北京 Linux 俱乐部
主要由北京市的大学生, 研究生，中科院、中关村等许多 Linux 的爱好者组成。 提供 Linux Today 中文版每日新闻。
南京 Linux 用户俱乐部
有不少大陆上的 Linux 资源连结。
上海 Linux 用户协会
上海 Linux 用户协会网页。
武汉自由软件协会
武汉自由软件协会网页。
重庆 Linux 俱乐部
重庆 Linux 俱乐部网页。
Chinese Linux User Group
Chinese Linux User Group 网页。
中国软件行业协会
国际自由软件应用研究发展分会网页。
Linux 公报中文版
Linux Gazette 的中译版本。由南京 Linux 俱乐部所支持。 是大陆方面主要的中文 Linux 计划。
中文 Linux 计划
收集了不少 Linux 的中文软件(GB 码)，文章。
The Linux Homepage by Homer
介绍 Linux 的站台，有不少 FAQ, HOWTO 的收集。

英文站台:

FRESHMEAT
一个以介绍 Unix, Open Source 软体为主的站台，每日更新。想知道软体的最新消息不可错过此站台。 上面还有 Linux 软体的资料库，方便大家寻找需要的软体。 谁说 Linux 上的软体不多? 看了这个站就知道 Open Source Community 的生产力惊人!
Slashdot: News for Nerds
Linux, Open Source 的及时新闻网，更新十分迅速。还有线上讨论区，成为网上最重要的 Geek/Nerd 聚脚地，有不少著名的 Hacker 都是其常客。
Open Source: the Future is Here
由 Eric S. Raymond 所主持，开放性原始码的总站台。有许多关於开放性原始码的文献资料。
Linux Weekly News
Linux 每周新闻。针对各主题分门另类介绍，包括核心，Distributions, 安全性，各平台移植等。每星期四更新。
LinuxFocus 线上杂志
Linux 的电子杂志。最值一提的是它有各语言的翻译。中文版的翻译计划正热列展开...:)
Linux Documentation Project
Linux 文件计划。收集了 Linux FAQ, HOWTOs, Man pages, 以及各种文件指引等。这里还收集了相当多的 Linux 连结及资讯。
LinuxHQ
介绍 Linux 核心为主的站台，包括新特徵的介绍，各版本的更新记录，非正式的修补(patch)收集，以及更新核心的注意事项等。
Linux Standard Base
发展及提倡一个 Linux 的基本标准，以增加不同 Linux 发行套件间的相容性。 同时帮助软体发展商移植并撰写 Linux 的软体。
Linux Reviews and Articles
□集了 Linux 文章以及各媒体上的 Linux 相关新闻。
