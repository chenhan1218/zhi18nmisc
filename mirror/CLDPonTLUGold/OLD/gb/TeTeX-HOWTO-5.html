5. LaTeX 扩充程式集与其他资源

上面提到，用划底线的方式来强调文件重点的方式，有一点特殊。实际上，对 TeX 　而言，划底线是数学上常用编排的方法，LaTeX 则可用下列的方法划底线： 

\underline{text to be underlined}
问题是划底线不会连接到下一行，因此在一些情况下，可能会不平顺的。然而， LaTeX 有一个现成的巨集，会在强调的部分划底线，它就是 ulem.sty，这是一个可以经由网际网路，自由传递及自由使用的巨集之一。

为了要使用 ulem.sty，可在前文使用： 

\usepackage{ulem}

那些提供LaTeX 使用的巨集指令还包括：

ifthen

在文件中包括有条件的陈述

initials

定义字首字母的字型

sanskrit

用　Sanskrit　编排文件

recipe

一个 LaTeX2e 清单式的命令

refman

不同的报告和文章格式

为了完整的给予目录中 URL 正确的位址和目录，可以把它分成主机和 CTAN archive 的目录，举例来说，位置 ftp.tex.ac.uk 的最高阶层的 CTAN 目录是 ctan/tex-archive. 使用 refman 巨集中表示完整的 URL 的目录就如下面一样： 

ftp://ftp.tex.ac.uk/ctan/tex-archive/   +
macros/latex/contrib/supported/refman   =

ftp://ftp.tex.ac.uk/ctan/tex-archive/macros/latex/contrib/supported/refman/
有些巨集不只一个档案，所以必须指明它所在目录的路径。

当你有 URL 时，便能从後面 Appendix A中任何一个 CTAN archive 位置，取回巨集。你也能从archive 的最高层的目录中，下载以 FILES.byname为名的档案，它□面有archive 全部档案的目录表，也可以在ftp中使用查询的命令： 

quote site index <keyword>
