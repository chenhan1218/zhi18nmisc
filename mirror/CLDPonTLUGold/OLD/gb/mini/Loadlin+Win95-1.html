1. 前言

本文件的目的在说明如何简单的利用加装 Loadlin.exe 程式於 Windows 95 开机选单以 F4 功能键自动执行 Linux 系统. 这个程序适用於 Windows 95 4.00.950, 4.00.950a, 及用FAT16 档案系统的 4.00.95b 版本, 及所有支援Loadlin 的 Linux 版本. 如果你习惯使用 LILO 开机管理程式, 请查询下述网站 Linux+Win95 Mini-HOWTO.

对於以 FAT32 档案系统的 Windows 95 4.00.950b 版本使用者, Loadlin 程式亦提供 了二种可以选择 Linux 或 Windows 95 分别开机的替代方法.
