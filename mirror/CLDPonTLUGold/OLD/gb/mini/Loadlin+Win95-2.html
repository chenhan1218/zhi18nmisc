2. 重要事项: 准备资讯及设定

2.1 对 Windows95 OSR2 (4.00.950b 版本) 使用者的警告事项:

千万 不可 於未详阅 Windows 95 OSR2 FAQ 的问题前试图按 以 Windows 95 双重开机. 该 FAQ 亦包含了许多对 FAT32 问题的探讨. 大略来说, 如果你虽拥有 OSR2 版本但仍然使用 FAT16 档案系统, 你仍可以使用 F4 功能键来作双重开机.

注意: 如果你装的是 Windows 95 OSR2 版本, 且使用 FAT32 档案系统, 则本 mini-HOWTO 所述以 F4 功能键来作双重开机的功能 无法使用而且切勿尝试.

但是, 针对此, 我们有一些於开机时可自动执行 Linux 的替代方式, 将留待 第 6 章 讨论.

2.2 设定前假定事项:

  • 你已成功的装妥 Windows 95 系统於你的电脑.
  • Windows 95 是装在电脑的 C 磁碟机.
  • 你已成功的装妥 Linux 系统於你的电脑.
  • 你知道 Linux 系统装在那一个分割区.
  • 你的硬碟中未安装 LILO 程式.
  • 如果你使用 Windows 95 OSR2 (4.00.950b 版本), 你已详读 Windows95 OSR2 FAQ.

2.3 在开始前须准备的物品:

  • 一张具备 DOS 5.0 或备有 config.sys 组态档的 DOS 6.x 系统档案 (io.sys, msdos.sys, command.com) 的开机磁片. 如果你使用 DOS 5.0 开机磁片, 你还需要 autoexec.bat 开机执行档. 如果你使用 Windows 95 OSR2 (4.00.950b 版本), 则你须要在 C 磁碟机的根目录中以文书编辑软体制作一个叫 Winboot.sys 的系统档. Winboot.sys 档案可以没有任何资料, 只要你有这个档案.

注意: 如果你的硬碟中已有 autoexec.dos, config.dos, io.dos, msdos.dos, 及 command.dos 等档案, 则你不须要准备开机磁碟片. 你可以於 C 磁碟机的根目录中键入: dir and dir /ah 来检查是否有上述档案. 此外, 如果你安装的是 Windows 95 的升级版本, 则你应该已有该等档案.

  • Loadlin.exe 程式档案
  • 你的 kernel 映射档案, 通常为 zImage 或 vmlinuz. (请参考 FAQ).
  • 简单的文书编辑软体, 像记事本 Notepad.
