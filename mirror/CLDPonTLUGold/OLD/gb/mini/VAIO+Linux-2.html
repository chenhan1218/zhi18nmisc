2. 硬体相容性

2.1 硬体和 Linux 不相容

在系统之中预先安装的大多数硬体是和 Linux 相容，然而有一些例外，如数据机， USB 接头和 MPEG 解码器， 请注意你不能使用数据机，它会对你们其中一些人会有问题 ， 装在 VAIO 之中的数据机是叫做 DSVD 数据机，它是 "Winmodem." 变型之一

(新加入: 12/28/98) 较新的型号好像是用标准的数据机代替 Winmodem。

2.2 部分相容的硬体

硬体中预先安装的音效卡和显示卡是部分相容的

音效卡

音效卡是使用微软公司声音系统 + 声霸卡和 Linux 相容 ，如果你要在 Linux 之下使用 MIDI，你需要 调整你的声音输入能力，针对这种情况，取消 声霸卡，如果你需要 MIDI，和声音输入，你可以考虑 买入 OSS/Linux，在 Linux 之下你可以使用它全部的功能， VAIO 的音效卡设备是 Yamaha OPL3SA， OSS/Linux 是 全部支援，它看起来像是 wavetable 规格中提到的综合体 且是全部使用软体达成，所以在这儿你不需要放弃硬体的能力 此时我写到这儿，它的版本是 是 OSS/Free 3.8，它支援 Yamaha OPL3-SA1 audio controller 之下的音效卡，请注意 2.0.x 之中不是预先安装的，你需要从 OSS 的网站下载， 2.1.x 跟著更新 OSS/Free 的版本而来，但是我没有测试它的相容性 ，如果你发觉到它可以在音效卡上工作 ，请让我知道。

显示卡

在 Linux 之下的显示卡是部分相容的，你 在 Linux console 模式之下使用是没有问题，如果你希望在 X Window 之下使用， 要取得 3.3.1或者是之後的 XFree86 ，它支援 ATI Rage，别的 商业的 X Window 系统也支援，无论如何， 3D 显示卡的 3D 加速功能在 Linux 之下没有作用， 对於 3D 加速显示卡 Linux 没有驱动程式 ， (更新 12/29/98)对於型号 505， XFree86 V3.3.3 或者以後的 ALSA 驱动程式是相容的。
