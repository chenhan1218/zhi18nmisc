8. RCS和 emacs(1)版本控制.

emacs(1)的版本控制便利性可用作是 RCS 的前端 .这资料运用特别地针对版本 19.34 的 GNU Emacs,在主要 Linux distributions 中有提供.当 使用emacs(1)编辑一个档案 它是使用 RCS 注册, 命令 vc-toggle-read-only (预设使用 C-x C-q )将会检查一个档案在 emacs 的版本控制中,然後 进入 RCS 中. Emacs 将会打开一个缓冲区在哪里你能够键入一个记录讯息 是被包含在 RCS 记录中.当你是完结了的键入一个记录 栏位,键入 C-c C-c结束你的输入和执行 登录的程序.

如果你有选择用 RCS 严厉的锁住档案,你必须 重新打开锁定的档案来编辑靠 emacs(1).你可以提 出档案用 emacs 版本控制使用那指令 % 在 buffer-menu 模式下.

更多资料,请看 GNU Emacs 手册和 Emacs info (信息)页.
