5. Windows 95 部分

在 win95 的 DOS 命令列□建个开机磁片。这很重要哦，是在 DOS命令列□做， 不是从 Windows 95内开个 DOS 视窗再做啊。拷贝 C:\WINDOWS95\COMMAND\SYS.COM 档 到这张开机磁片□ (如果你的路径不同就改一下吧)。然後重新由这张磁片开机进入命 令列。执行 

SYS C:
这应该可以把 Windows NT OS loader 移除，就可以从 LILO 开机到 Windows 95 了。

现在你可以杀掉 NT loader 的几个档案: 

Boot.ini
Bootsect.dos
Ntdetect.com
Ntldr
