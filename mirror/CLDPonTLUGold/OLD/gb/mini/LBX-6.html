6. 我需准备要什麽去使用 LBX？

你需要一个已经将 LBX 编译整合在本端的 X Server。 除非你在编译的时候指定不将 LBX 整合於其中， 否则 X11R6.3 会自动的使用 LBX 的特性。 所有的 XFree86 3.3 server 预设都会把 LBX 整合於其中。

你可以使用 xdpyinfo 这个命令检是你的 server 是否打开了 LBX 的功能。 执行 xdpyinfo 即可在 number of extensions 的列表下面，见到 LBX 列在其中。

下一不你需要一个在远端已经将 lbxproxy 这程式编译在内的系统。 这边有个陷阱。 如果远端的系统和你本端不同，那麽在本端的 lbxproxy 将不会作用。

很不幸的，没所所谓分离开的 lbxproxy ，所以 你必须 (a) 所有的远端系统都使用X11R6.3，或是 (b) 找一个针对你的本端系统预先编译好的 lbxproxy。 当然後者是比较方便的。

lbxproxy 仅仅是一个单一执行档。 它没有任何的设定档或是资源档，或诸如此类的档案。
