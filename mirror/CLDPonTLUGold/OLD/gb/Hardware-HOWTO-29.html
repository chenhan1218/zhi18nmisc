29. 词汇/术语

AGP

Accelerated Graphics Port 加速绘图埠。是个汇流排互连机制，设计来改善 3D 三度空间绘图应用。 AGP 是从绘图子系统到内部逻辑晶片组间的专用汇流排 http://www.euro.dell.com/intl/euro/r+d/r+dnews/vectors/vect_2-1/v2-1_agp.htm

ATAPI

AT Attachment Packet Interface. 是新的协定用来控制大量储存设备，和 SCSI 协定很像。他是在 ATA (AT Attachment) 介面上所建立的，正式的 ANSI 标准名称为- 发展给硬碟使用的 IDE 介面。 ATAPI 通常运用在硬碟、 CD-ROM 磁碟机、磁带机、以及其他设备上。

ATM

Asynchronous Transfer Mode 不同期传输模式

CDDA

?? CD-ROM/Writer 读取音轨的能力。

DMA

Direct Memory Access 直接记忆体存取

EGA

Enhanced Graphics Adapter 加强型绘图卡

EIDE

Enhanced IDE 加强型 IDE

EISA

Extended Industry System Architecture 扩充的工业系统架构

FDDI

Fiber Distributed Data Interface. 光纤分送资料介面，是高速环型区域网路。

IDE

Integrated Drive Electronics. 每一个磁碟机都有个内建的控制器。

ISA

Industry System Architecture 工业系统架构

ISDN

Integrated Services Digital Network 整合服务数位网路

MCA

MicroChannel Architecture 微通道架构

MFM

Modified Frequency Modulation 被修改的调变频率 ?

MMX

Multimedia Extensions 多媒体延伸指令集。新加到Intel Pentium 处理器□面的指令集。它提供更佳的音效与影像品质。

PCI

Pheripheral Component Interconnect 周边元件互连。由 Intel设计的32-位元汇流排。详情可在 http://www.pcisig.com 和 http://infoserver.ee.siue.edu/~jbutter/EE580_1.html□面找到。

RAID

Redudant Arrays of Inexpensive Disks 不贵的过剩磁碟机阵列。RAID 的基本概念来自於集结多个容量小、又不贵的磁碟机成为一个磁碟阵列。产生比一个容量大但是较贵的磁碟机还要好的效能。现在共有五种不同型式的过剩阵列架构；从 RAID-1 到 RAID-5。非过剩磁碟阵列也可称为 RAID-0。
http://www.uni-mainz.de/~neuffer/scsi/what_is_raid.html

RLL

Run Length Limited 限制执行长度 ?

SCSI

Small Computer Systems Interface 小型电脑系统介面。定义来给电脑使用的设备小型架构。它可以让你采用单一个转接卡来连接全部的设备。
http://www.uni-mainz.de/~neuffer/scsi/what_is_scsi.html

SVGA

Super Video Graphics Adapter 超视讯绘图转接卡

UART

Universal Asynchronous Receiver Transmitter 通用异步接收/传送器

VGA

Video Graphics Adapter 视讯绘图转接卡

VLB

VESA Local Bus

WORM

Write Once Read Many
