1. 介绍

这是 SMB HOWTO {如何使用 SMB} . 这文件描述在 Linux 上如何使用 Session Message Block (SMB) protocol, 也叫做 NetBIOS 或 LanManager protocol.

本文件的维护者是 David Wood ( dwood@plugged.net.au). 再者,修改或修正可以寄到那,以便下一版包涵进去.

SMB protocol 被使用在 Microsoft Windows 3.11, NT 和 95 用来分享磁碟及印机. 使用 Andrew Tridgell 所做的 Samba 工具套件, UNIX (包含 Linux) 机器能分享磁碟与印表机给 Windows 主机.

吾人使用 Samba 有四样东西可以做:

  1. 分享 linux 磁碟给 Windows 机器.
  2. 分享 Windows 磁碟给 linux 机器.
  3. 分享 linux 印表机给 Windows 机器.
  4. 分享 Windows 印表机给 linux 机器.

这些东西都已包含在本文件中.
