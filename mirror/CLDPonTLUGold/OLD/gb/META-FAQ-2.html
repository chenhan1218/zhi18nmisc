2. 取得 Linux

2.1 Linux FTP 站台

最完整的 Linux FTP 站台列表是在 Linux INFO-SHEET，可从 http://sunsite.unc.edu/LDP/HOWTO/INFO-SHEET.html 取得． 最重要的站台列在这里; 请看 INFO-SHEET 寻找离你最近的站台(有许多的映射站)． 

textual name             numeric addr    Linux directory
=======================  ==============  ===============
tsx-11.mit.edu           18.86.0.44      /pub/linux
sunsite.unc.edu          152.2.22.81     /pub/Linux
ftp.kernel.org           206.184.214.34  /pub/linux

这些站是大部份上传档案的``主''站台．有许多的映射站; 请使用离你最近(网路上的距离)的站台．

2.2 物质媒体上的 Linux

Linux 也以物质媒体形式散布，主要是由一些厂商制作的 CD-ROM． 请参阅定期发布在 comp.os.linux.announce 的 Distribution HOWTO，也可从 http://sunsite.unc.edu/LDP/HOWTO/Distribution-HOWTO.html 取得．

2.3 AFS

Linux 可挂上 sipb.mit.edu 的目录 project.linux 由 AFS 取得．

2.4 商业网路

Compu$erve 有一些 Linux 档案．

2.5 电子邮件伺服器等

Sunsite 提供 ftp-mail 服务 --- 寄电子邮件到 < ftpmail@sunsite.unc.edu>．
