17. 可移式磁碟机 (抽取式磁碟机)

只要控制卡有被支援，所有的 SCSI 磁碟机应该都可以使用，包含 光磁碟片 (MO), WORM, floptical, Bernoulli, Zip, Jaz, SyQuest, PD 等等的。

  • Panasonic MO (合并了 CD-ROM 磁碟机和可移式光学磁片的功能)。在建构 kernel 时需要打开某个开关才能同时使用这两个功能。
  • 并列埠 Zip 磁碟机
    ftp://gear.torque.net/pub/
  • 并列埠 Avatar Shark-250
    http://www.torque.net/shark.html

抽取式磁片就像硬碟和软碟片般地来使用，只需 fdisk/mkfs 和 mount 磁碟机即可。 Linux 提供了磁碟机锁定功能，只要你的碟磁机有支援的话。如果磁片是 MS-DOS 格式的话，那麽mtools 也可以被使用。

CD-R 磁碟机需要特殊的软体才能使用，请参阅 CD-R Mini-HOWTO。

Linux 有支援 512 和 1024 bytes/sector 的磁片。从 kernel 2.1.32 开始 Linux 也支援了 2048 bytes/sector的了。给 kernel 2.0.30 用的修补档可以在下面取得。 http://liniere.gen.u-tokyo.ac.jp/2048.html

如果使用

  • Fujitsu magneto-optical disk drives M2513
的话，你会需要 2048 bytes/sector 的支援才可。

从 pre-patch-2.0.31-3 开始，IDE/ATAPI 内接式 Zip 磁碟机， flopticals 及 PD 都已被支援了。

  • LS-120 floptical
  • PD-CD
