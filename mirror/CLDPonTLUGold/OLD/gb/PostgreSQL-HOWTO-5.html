5. 何处取得 ?

你可从以下地方购买 Redhat Linux 光碟、Debian Linux 光碟或 Slackware Linux 光碟，从中找到 PostgreSQL 套件（原始码和二元码）∶

  • Linux System Labs 网址∶ http://www.lsl.com/ （7 美元）
  • CheapBytes Inc 网址∶ http://www.cheapbytes.com/ （7 美元）
  • Debian 主网址∶ http://www.debian.org/vendors.html

PostgreSQL 组织也售卖‘PostgreSQL CDROM’，内含完整的原始码，许多 Unix 操作系统的可执行档及完整的文件。

  • 从主网页取得 PostgreSQL 光碟∶ http://www.postgresql.org 30（美元）

PostgreSQL 只含二元档的发行版本 (distribution)∶

  • PostgreSQL RPMs 的维护人是 Lamer Owen，在 lamar.owen@wgcr.org 找到。
  • PostgreSQL 源 RPM 和二元 RPM http://www.ramifordistat.net/postgres。
  • PostgreSQL 源 RPM 和二元 RPM http://www.postgresql.org 。点选“Latest News”和 Redhat RPMs。
  • PostgreSQL 源 RPM 和二元 RPM http://www.redhat.com/pub/contrib/i386/ 。ftp 站台在 ftp://ftp.redhat.com/pub/contrib/i386/
  • Solaris, HPUX, AIX, IRIX, Linux 的二元档站台∶ ftp://ftp.postgresql.org/pub/bindist
