19. 数据机

所有连接到串列埠的内接式数据机或外接式数据机都可以工作。哎，有些制造商只制造仅相容於Windows 95 的数据机。 你可查看附录 D 有所有不相容 Linux 的硬体。

有少部份数据机的 DOS 软体会在执行时期下载控制程式。它们能以在 DOS □面载入程式後，再热开机的方式来使用它们。这类的数据机最好不要用，因为以後在非 PC 硬体上就不能用了。

所有的PCMCIA 数据机应该都可在 PCMCIA 驱动程式下使用。

传真数据机需要适当的传真软体才能使用。另外，请确认数据机的传真部份是否支援 Class 2 或 Class 2.0。 似乎大部份在 unix 上的传真软体只支援 Class 1.0 都不能用是真的。
