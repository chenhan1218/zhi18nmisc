5. Shells 设定

请参阅 ISO 8859-1 HOWTO 以获得更多资讯.

5.1 bash

建立一个档案 $HOME/.inputrc 并在内容加入以下句子:

-----
set meta-flag On
set convert-meta Off
set output-meta On
-----

5.2 tcsh

在 $HOME/.login 或 /etc/csh.login 中加入以下定义: setenv LANG iw_IL.ISO8859-8 (or iw_IL) 事实上, 因为 binary version 的 tcsh 并不遵守 nls, 你可以将 LANG 设为任何东西 (不需要 /usr/lib/nls...) 而不会有任何问题. lang.name 并会显示我的 Digital bias...
