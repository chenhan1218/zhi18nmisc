4. 其它

4.1 除了翻译，我还能做什麽?

答: 您还可做这些:

  • 协助建立翻译常用辞汇。请参考 翻译小辞汇 的说明。
  • 帮忙找出 CLDP 文件的打字，排版等错误。
  • 提供 FTP 机器做为本中译计画的正式站台。 如果您愿意的话，请注意您的机器最好是跑 Linux，而且必须够新，够快，够稳(不能常 crash)，最重要的是，必须要有足够的网路频宽! 还有这机器必须能长久存在。
  • 提供机器 Mirror 这些中译文件。Mirror 的站台就没有上面那些要求了，只要您愿意就好!
  • 为本网页设计 LOGO, 加上一些美丽的图片吧!
  • 任何的建议，批评，点子，or WHATSOEVER。
  • 提供金钱赞助，数目随意……

4.2 我想翻译/写作其它与 Linux 相关的文件，可以放在 CLDP 网页上吗?

答: 是的! 相当欢迎。将它寄给我即可。不过因为要放上网页，请以 HTML 或其它易於转换为 HTML 的格式写作。

4.3 除了翻译 Linux HOWTOs, CLDP 可还有其它的计划?

答: CLDP 目前还没有其它具体的计划。但相当欢迎有人能提出新计划的想法与建议。 例如翻译线上手册(man pages), 翻译 Linux Journal, 中文 Linux 每日新闻等都是不错的点子。

一个计划必须有明确的方向与详细的执行步骤才能有所成。因此如果您有好的想法，不妨来信和我讨论。 只要能提出具体可行的计划，相信必能引吸许多 Linux 高手的参与。

4.4 你是谁?

答: 我是 CLDP 计划的发起人，也是目前计划的协调人。我负责接收各位译者寄来的文件，转换後发表在网页及其它的地方。 我也维护了 HOWTO INDEX 的中译版，以及 Linux Chinese HOWTO 等文件。

我也对任何 Linux 中文化或推广 Linux 使用的活动有兴趣。如果您有这些想法或建议，欢迎来信。
