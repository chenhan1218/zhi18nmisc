6. Morse Code -- 莫尔斯电码类

使莫尔斯电码通信变得容易使用的软件集。

6.1 Pileup -- 嘈杂的 CW 训练器

作者

Richard Everitt G4ZFE richard@babbage.demon.co.uk. 参考了 Steve Merrifield 写的 sccw。

简单说明

CW pileup 训练程序。

当前开发状态

版本：1.0。运行稳定且有进一步的开发计划。

更新日期

1997年4月1日

系统需求

支持 Linux 的声卡。

详细说明

基于 sccw 的 CW pileup 训练器 (类似于 PED)。最多允许 9 个电台以不同的速度、音调、音量同时工作。为增加真实性，呼叫的电台呼号是从 1996 CQ WW CW 竞赛 M6A 的日志中摘录下来的。

我把 /usr/dict/words 作为训练文件，文件中有一些令人发笑的单词。所以如果你并不练习莫尔斯电码，也会觉得很有趣。

从何处可取得本软件。

你可以从他的主页： Pileup Home Page 也可以从 sunsite.unc.edu 取得 pileup 软件包。

使用许可/版权

GNU GPL

软件提供者：

Richard Everitt G4ZFE richard@babbage.demon.co.uk.

6.2 bip -- 类 Linux 系统的莫尔斯电码练习器

作者

Marcin Skubiszewski, Marcin.Skubiszewski@inria.fr. 24 rue de l'Oasis, 92800 Puteaux, France

简单说明

一个学习莫尔斯电码的程序。(运行在多种类型的 Unix 平台上)

当前开发状态

产品已发行

更新日期

1997年4月1日

系统需求

能运行在多种类型的 Unix 平台上，在 Linux 下，有没有 X11 和声卡均可。

详细说明

bip 作为一个电码练习程序提供了广泛的功能。它能读取文本文件或随机 产生练习字符，其他特点如下：

  • 符合 ARRL 标准的莫尔斯代码。
  • 音量、音调可调。
  • 速度可调，包括 Farenheit 调整方法
  • 模拟手键的莫尔斯发报。

6.3 cw2hex -- CW 到二进制的转换程序

作者

G. Forrest Cook, WB0RIO, cook@stout.atd.ucar.edu

简单说明

cw2hex 能为电码发生器的 EPROM 将文本文件转换成十六进制文件。

当前开发状态

版本 1.1 测试完成，准备发行。

更新日期

1997年4月1日

系统需求

单独的文本程序，只须 GCC 编译器。

详细说明

本程序把字符输入转换成标准的莫尔斯十六进制文件。把此文件输入到 EPROM 里，并把 EPROM 连接到一个简单的电路中，此电路就能发出预先设定好的莫尔斯电码。常用于 CW 呼叫器或 CW 信标中，本软件还附带一个可在此软件下工作的电子键电路。

从何处可取得本软件。

你可以从： www.atd.ucar.edu 取得 cw2hex

使用许可/版权

GNU 版权

软件提供者：

G. Forrest Cook, WB0RIO

6.4 SoundCard CW -- 使用声卡的 CW 练习器

作者

Steve Merrifield, VK3ESM, sjm@ee.latrobe.edu.au

简单说明

Linux 上的莫尔斯电码练习器, 使用声卡。

当前开发状态

开发完成

更新日期

1997年4月1日

系统需求

有 FM 模式的支持 Linux 的声卡。

详细说明

本程序使用声卡来播放莫尔斯电码。它并不依赖 Linux 虚拟控制台声音技术，所以它能运行在 X11 上，它具有以下特点：

  • 频率可调
  • 音量可调
  • 速度可调
  • 延迟可调
  • 每组字符个数可调
  • 可产生任意组
  • 有解码功能，并能播放文本文件
从何处可取得本软件。

本软件可以从它的主页取得： livewire.ee.latrobe.edu.au。

伯克利式的版权，需要得到作者许可。

Steve Merrifield, VK3ESM

6.5 GW4PTS Morse trainer. -- 莫尔斯电码练习器

作者

Alan Cox, GW4PTS, alan@lxorguk.ukuu.org.uk

简单说明

使用 PC 机内置扬声器的莫尔斯电码练习器

当前开发状态

运行稳定, 工作正常，还未最后完成。

更新日期

1997年4月1日

系统需求

任意版本的 Linux。

详细说明

Alan 只花了一个小时写这个程序。程序做得很小巧，允许你使用 PC 内置扬声器以不同速度和频率练习莫尔斯电码。你可以指定从键盘输入，文件输入，或任意产生字符组练习。运行时你必须从虚拟控制台唤醒该进程，并依赖一些内核调用来产生声音，所有这些从一个 XTerm 运行并不是很容易。在它练习某个字的莫尔斯电码时，在屏幕上同时显示该字符的发音（Di, Dit, Dah 等）。Alan 希望有人完成并增强他在源代码注释中所列举的功能。
