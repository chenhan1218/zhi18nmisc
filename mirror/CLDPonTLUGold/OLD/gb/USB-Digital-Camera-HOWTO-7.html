7. 不要苦恼

更改SCRIPT FILE 可能会使你执行出错,现在不理它们. 在你苦恼并抱怨它们没有效果之前,看一下你的camera有没有pictures. 如果你看过了, 请将 directory path names (路径名称)写下来. 回到文档编辑器中,将 dcim/100msdcf/*.jpg 替换之. 你可能需要用几个不同的 script files 来处理不同的 picture 格式.

现在你的硬碟机的文档目录和 Section A.2 相似.执行并确认它的正确性:

bash $ dir -R name-of-your-picture-directory
