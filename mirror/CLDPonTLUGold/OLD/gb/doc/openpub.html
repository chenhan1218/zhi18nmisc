I. 未修改与修改版本两者的需求

公开发行刊物可能完全或一部份，在任何媒体，如物质媒体或电子媒体被复制散布使用，倘若在发行之前能加入这份授权书，或是在重制时将这份授权书纳入成为相关文件 ( 原著作者在散布之前可选择此份授权书中其所需要的选项 )。

在作品中要加入这份授权书，适当的参考型式如下:
Copyright (c) <year> by <author's name or designee>. This material may be distributed only subject to the terms and conditions set forth in the Open Publication License, vX.Y or later (the latest version is presently available at http://www.opencontent.org/openpub/).

Copyright (c) <年> by <著作人的名称>。 此产品内容只能在遵守此份授权书下被发布与在公开发布授权书之前设定限制条件，遵守  vX.Y 版或公开授权书更新的版本 ( 取得这份授权书的最新版本在 http://www.opencontent.org/openpub/ )。

相关的参考资料在後续的任何选项必须立即的由文件的作者或出版者作选择 ( 看第 VI 节 )。

公开发布授权书的商业上重新分配产品内容是允许的。

任何发行刊物在一般 ( 纸 ) 的书必须要引用原本的作者或著作人的姓名。作者或著作人的姓名将会显现在书的封面上。 所有书的表面原始出版者的工作头衔应该被广泛的显示与相关的贡献人士应尽可能的顾虑到并显示其头衔。


II. 版权

每一个公开发行刊物的版权为著作人或设计者所拥有。


III. 授权书的有效□围

下面的授权项目适用公开发布工作者，除非文件有其他明确的指示。

授权项目仅适用於公开发布工作的集合或一部份的公开发布工作与其他的工作或程式在相同的媒体，这份授权书将不套用到那些其他工作上。这些工作将包含一个介绍说明来指定公开发布具体的与适当的版权介绍。

SEVERABILITY  如果这份授权书的任何部分在各种权力上是被找到有未规定的，保持授权书的这部份来达到更好的规□。

无担保  公开发布工作者是获准与提供 "也就是" 不担保任何种类，表示或意指， 包含，但不被限制，这意指可买卖的担保与适当特殊用途或没有违反的担保是被允许。


IV. 修改者需求

所有文件修改版本都被适用授权，包含翻译版本，名言集，编辑与部分文件，必须满足下面的需求:
  1. 修改过的版本必须要标示以下事项。
  2. 个人所修改的部分必须能被适别并打上修改日期。
  3. 容许原本作者与出版者如果可适用的必须依据一般理论的引用习惯被保留。
  4. 原本未修改文件的取得位置必须能被知道识别。
  5. 原本作者 ( 或许多作者的 ) 名称在没有原本作者的 ( 或许多作者的 ) 许可下并不被使用来声明或暗指文章结果的背书保证。

V. 好的-习惯推荐

为了达到这份授权书的要求，重新分布时强烈的要求与建议如下:
  1. 如果你是使用硬体复制或 CD-ROM 散布公开发行刊物工作内容，不论是用手写或是用媒体转载，在三十天之前你必须利用电子邮件通知作者说你想要重分配，给作者预备时间来更新文件。不论改变文件多少部分，这份通知应该说明修改部分。
  2. 所有独立修改 ( 包含删除 ) 必须在文件中明显地标记或是在文件的附件中说明。
  3. 最後，公开发布授权工作者应考虑好的方法来提供任何自由的硬体复制与 CD-ROM 的复制，在这授权书之下做这件事并非义务的。

VI. 授权书项目

使用公开发行授权书刊物的作者 ( 或许多作者 ) 与/或 出版者可以藉由增加说明至参考文件的方式或复制这份授权书的方式来选择特定选项。这些选项是考虑到部分的授权情况，与考虑到这份授权书在推演後所需包含的内容 ( 或是参考部分的内容 )。

A.  禁止分发散布没有作者述说清楚授权许可的独自修改版本。 "独自修改" 的定义是改变文件与语意有关的内容，排除只有在格式上或印刷上更正的修改行为。

为了达到此目的，增加以下这句话到授权书内的参考部分或复制部分 `Distribution of substantively modified versions of this document is prohibited without the explicit permission of the copyright holder.' ( 禁止分发散布没有作者述说清楚授权许可的独自修改版本 ) 。

B.  此工作或衍生工作中的任何发行刊物的所有或部分内容，禁止在一般 ( 纸 ) 的书的商业交易，除非之前获得版权所有人许可。

为了达到此目的， 增加以下这句话到授权书内的参考部分或复制部分 'Distribution of the work or derivative of the work in any standard (paper) book form is prohibited unless prior permission is obtained from the copyright holder.'  ( 此工作或衍生工作中的任何发行刊物的所有或部分内容，禁止在一般 ( 纸 ) 的书的商业交易，除非之前获得版权所有人许可 ) 。
