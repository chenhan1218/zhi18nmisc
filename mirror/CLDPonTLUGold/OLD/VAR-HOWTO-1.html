1. 關於這份文件

這是 Linux VAR HOWTO。它包含了一份不製造硬體或製作套件軟體、而是將已有的產品改進並加入新的價值的服務公司名單。 所謂附加的價值有很多種型態，比如說安裝 Linux 到電腦的硬體上，建構或依客戶要求所做的應用軟體。 然而，目前這份 Linux VAR HOWTO 只包含那些以跑 Linux 為目的而先建構好的硬體之銷售的公司。 除此之外，有提供其他服務的公司，我們都歡迎您與這份文件的維護者 Martin Michlmayr <tbm@cyrius.com>連絡. 我們會把你的公司也列入這份名單點。

如果你與下列任何的一家公司連絡，請順便提一下這份 Linux VAR HOWTO。

如果你需要了解更多的事情 Linux Documentation Project 或是關於 Linux HOWTO 的，不要客氣，請和 連絡人 Tim Bynum <linux-howto@sunsite.unc.edu>連絡。

Greg Hankins 會每個月定期將這份列表貼到許多國家以及國際性的新聞討論群媕Y。 除此之外， Linux VAR HOWTO 可以經由 WWW 的 http://var.cyrius.com獲得。 新版的 Linux VAR HOWTO 總是會第一個被擺到 這上面，所以請確認你所閱讀的是最新的版本 !
