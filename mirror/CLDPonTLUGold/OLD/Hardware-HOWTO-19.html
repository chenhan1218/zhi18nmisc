19. 數據機

所有連接到串列埠的內接式數據機或外接式數據機都可以工作。哎，有些製造商只製造僅相容於Windows 95 的數據機。 你可查看附錄 D 有所有不相容 Linux 的硬體。

有少部份數據機的 DOS 軟體會在執行時期下載控制程式。它們能以在 DOS 堶掘入程式後，再熱開機的方式來使用它們。這類的數據機最好不要用，因為以後在非 PC 硬體上就不能用了。

所有的PCMCIA 數據機應該都可在 PCMCIA 驅動程式下使用。

傳真數據機需要適當的傳真軟體才能使用。另外，請確認數據機的傳真部份是否支援 Class 2 或 Class 2.0。 似乎大部份在 unix 上的傳真軟體只支援 Class 1.0 都不能用是真的。

    ftp://sunsite.unc.edu/pub/Linux/apps/serialcomm/smdl-linux.1.02.tar.gz
  • Motorola MODEMURFR 內接式 56K. 新增一行到 RC.SERIAL 來要求一個 IRQ 以及連接埠，如果他們是非標準的。
  • ZyXEL U-1496 series - ZyXEL 1.4, modem/fax/voice 控制程式

  • ZyXEL Omni TA 128 - modem/fax/voice 控制程式
    http://www.pe1chl.demon.nl/ZyXEL/ZyXEL-1.6.tar.gz
