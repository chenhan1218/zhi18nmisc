7.2 美國

有種種理由應該使區域性的 Linux 使用者協會成為正規組織。不過在這裡我不想多加說明。但是，如果你想正式組織你們本地的使用者協會， 本節提供一些有用的資料。

注：本節內容不應作為法律問題的依據。具體法律問題需要征求律師的諮詢意見。

在美國，區域性的 Linux 使用者協會至少可選用兩種不同的法律地位：

  1. 組成非盈利機構
  2. 免稅

雖然每州的有關法規不同，但大都允許使用者團體組成非盈利機構。區域性的 Linux 使用者協會組成一個機構的好處是依法限制了協會成員和自願人員的法律賠償責任， 也限制或豁免了該州征收的公司稅。

在把你們的 Linux 使用者協會組織成非盈利機構之前應該征詢律師的意見。但如果在同律師會面之前， 先對問題取得一些了解，可能會減少律師的費用。我建議可參考 Non-Lawyers Non-Profit Corporation Kit (ISBN 0-937434-35-3)

至於免稅問題，這不是法律地位問題，而是稅務局的判斷問題。應該知道， 組成非盈利機構并不保證稅務局豁免你們協會的稅金。一個非盈利機構但不能免稅， 這是完全可能的。

美國稅務局有相當簡單的文件解釋申請免稅的準則和程序。 可向稅務局索取Publication 557: Tax-Exempt Status for Your Organization。這份文件也作為 Adobe Acrobat 文檔，可在稅務局的網址上調閱。我堅信在申請成為非盈利機構之前， 先閱讀這份文件。成為非盈利機構不能保證稅務局會認為你們的LUG屬於免稅機構， 但組成機構的方式不同卻有可能使稅務局無法將你們的LUG當成免稅機構對待。 Tax-Exempt Status for Your Organization中附有說明，詳細載列成為免稅機構的條件。

最後，網際網路上也有資料，或許可供你作為組成非盈利機構或使其具有免稅資格的參考。

7.4 政治問題

Chris Browne指出下列在 LInux 使用者協會內部時常發生的問題。

人們對自由軟件具有不同的看法

Linux 的使用者形形色色、各種人物都有，只要這些人聚在一起， 必然會有一些問題發生。有些近乎是政治激進派，他們認為所有軟件都應該“免費”， 不應有任何例外。因為 Caldera 出售它們的 Linux 版本，而且 收費不贄，並且不肯把盈利捐給(選一個心目中喜愛的機構)，因此它們必然是“壞蛋”。Red Hat 和 S.u.S.E 也如出一轍。 但這三個公司卻一向大力支助自由軟件，從未停頓。

另一批人可能認為他們有辦法為他們的喜好和利益把 Linux 平台的“免費”特性發揮得淋浰盡致。 許多使用 BSD UNIX 改進版本的用戶認為他們的許可証雖然的確允許公司行號根據他們的 操作系統建立公司自用的系統， 但卻想運用于 Linux 的 GPL 的“強制永久免費”。 不要認為所有鼓吹這種論點的人都是貪得無厭之徒。

這些人聚在一起，就會產生不同意見。

負責協會會務的人員必需明瞭下列事實：

  • 目前對 GPL 有各種看法，對落實 GPL 的方式也莫衷一是。因此對 GPL 和 其他各種許可証的使用方式很容易導致誤解。
  • 多機構都捐助 Linux，因此才能提供 Linux 的免費下載，尤其是通過這種下載能鼓勵更多 人參與 Linux，那就可能會有更多人提供捐助。
  • 對 Linux 的許多重大捐助都來自公司企業。看 看 Linux 的原始碼和 XFree86 和 GCC 等重要輔助系統，許 多捐助都是公司企業提供的。
  • 上市的產品不一定“較好”，但也不一定“差勁”。

爭論的要點遠不只這些；電腦“聖戰”早已拉開序幕，Linux 及其他 UNIX 系統或視窗的各種 操作系統孰优孰劣、“IBM PC”和各種 Motorola 68000 系統何者性能更強、1970年代各種系統的 優劣以及 KDE 對 GNOME 的問題，早已是爭論不休的問題。

LUG 社團的負責人員應該設法緩和這種歧見，而不應加劇分裂。LUG 的負責人員 必須巧妙地運用厚黑學。

未來必然會有意見相左的時刻，負責人員必須能夠排解糾紛，而不要使問題更形惡化。

非盈利機構的財務來往

在一個非盈利機構中需要非常注意財務的來往。商場上，利潤一向都是追求的目標， 大家并不太錙銖計較用錢的方法。

在一個非盈利機構中，情況就不如此單純。有些人十分把持原則，事事都可能小題大作。 參加會議討論財務問題的人數愈多，為雞毛蒜皮的事引起爭論的機會也愈多。

Linux 使用者協會最好不收會費，否則就有一項具體事務需要時時回應會員提出的質訊。 不收會費就不存在濫用會費的問題。

如果社團或協會經費充裕，就得有人專門管理財務，向會員報告支出情況。

一個活動頻繁、成長發展的協會，應有一人以上專管此事。一個弊病叢生的非盈利機構的財務 狀況時常會被一些不肯放手的主管人員把持。理想狀況是 Linux 使用者協會的工作劃分應由 一些輪調，尤其是管理財務的工作。

財務報告應定期提供給想要了解財務情況的會員。 維持在稅法上正規“慈善機構”地位的 Linux 使用者協會每年至少得向本地稅務機關保稅一次， 起碼利用這個機會可使會員了解協會的財務狀況。

根據 Linux 編寫的財務軟件日益增多，現在應該已經佷容易定期製作財務報表。隨著網際網路的發展， 這些報表甚至可通過 WWW 提供。
