2. 取得 Linux

2.1 Linux FTP 站台

最完整的 Linux FTP 站台列表是在 Linux INFO-SHEET，可從 http://sunsite.unc.edu/LDP/HOWTO/INFO-SHEET.html 取得． 最重要的站台列在這裡; 請看 INFO-SHEET 尋找離你最近的站台(有許多的映射站)． 

textual name             numeric addr    Linux directory
=======================  ==============  ===============
tsx-11.mit.edu           18.86.0.44      /pub/linux
sunsite.unc.edu          152.2.22.81     /pub/Linux
ftp.kernel.org           206.184.214.34  /pub/linux

這些站是大部份上傳檔案的``主''站台．有許多的映射站; 請使用離你最近(網路上的距離)的站台．

2.2 物質媒體上的 Linux

Linux 也以物質媒體形式散佈，主要是由一些廠商製作的 CD-ROM． 請參閱定期發佈在 comp.os.linux.announce 的 Distribution HOWTO，也可從 http://sunsite.unc.edu/LDP/HOWTO/Distribution-HOWTO.html 取得．

2.3 AFS

Linux 可掛上 sipb.mit.edu 的目錄 project.linux 由 AFS 取得．

2.4 商業網路

Compu$erve 有一些 Linux 檔案．

2.5 電子郵件伺服器等

Sunsite 提供 ftp-mail 服務 --- 寄電子郵件到 < ftpmail@sunsite.unc.edu>．
