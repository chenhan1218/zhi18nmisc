6. 其它

6.1 有用的資源

  • IP Masquerade Resource page 應有更多建立 IP Masquerade 的資訊。
  • IP masquerade mailing list archive 包含最近送到通信論壇上的信件。
  • 給核心 2.2.x 與 2.0.x 的 Linux IP Masquerade mini HOWTO。
  • 如果你用更舊的核心，請看 IP Masquerade HOWTO for kernel 1.2.x
  • IP masquerade FAQ 有某些一般的資訊。
  • Linux IPCHAINS HOWTO 及 http://www.rustcorp.com/linux/ipchains/ 有許多關於 ipchains 使用的資訊，並有 ipchains 的原始碼及二進位執行檔。
  • X/OS Ipfwadm page 包含 ipfwadm 套件的原始碼，執行檔，文件及其它的資訊。
  • 由 Lee Nevo 所維護的網頁 可與 Linux IP masquerading 運作的應用程式 提供提示與技巧使得應用程式能與 IP Masquerade 運作。
  • The LDP Network Administrator's Guide 這是新手嘗試設定網路的必要資訊。
  • Trinity OS Doc, 一份關於 Linux 網路相當容易理解的指引。
  • Linux NET-3 HOWTO 也有許多關於 Linux 網路方面有用的資訊。
  • Linux ISP Hookup HOWTO 以及 Linux PPP HOWTO 給你如何把 Linux 主機連上網際網路的相關資訊。
  • Linux Ethernet-Howto 有關設定執行乙太網路的區域網路很棒的資訊來源。
  • 你也可能對 Linux Firewalling and Proxy Server HOWTO 有興趣。
  • Linux Kernel HOWTO 將會指引你編譯核心的過程。
  • 其它的 Linux HOWTOs 像是 Kernel HOWTO。
  • 張貼到這個 USENET 新聞群組: comp.os.linux.networking

6.2 Linux IP Masquerade 資源

Linux IP Masquerade Resource 是一專門用來提供 Linux IP Masquerade 資訊的站台，也是由 David Ranch 及我來維護。 這邊有關於 IP Masquerade 最新的資訊，可能也有未包含在本文件的其它資訊。

你可在下列地點找到 Linux IP Masquerade Resource:

  • http://ipmasq.cjb.net/, 主站台，重導至 http://www.tor.shaw.wave.ca/~ambrose/
  • http://ipmasq2.cjb.net/, 第二分站, 重導至 http://www.geocities.com/SiliconValley/Heights/2288/
