這份文件探討從 Linux 系統當機中回復的方法。造成 Linux 系統當機的各種原因可能是 - LILO 毀損、 linux 無法開機、 硬碟主開機記錄(Master Boot Record, MBR)損壞、或是安裝其他作業系統例如 Windows NT 而將 LILO 或 MBR 蓋寫掉所造成 linux 無法開機。
