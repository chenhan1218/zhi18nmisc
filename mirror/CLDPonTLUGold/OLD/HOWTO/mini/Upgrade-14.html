14. With the new linux system booted from the hard disk, edit /etc/fstab

and add your swap partition. Then run the command "swapon -a". I don't know why, but Slackware setup doesn't offer to do this for you if your swap partition exists already. Then, when you boot your new system and the rc.S script tries to turn swapping on, it can't find the partition in the fstab file and swapping doesn't get enabled. This step fixes it.
