6.1 TroubleShooting

Q: I get an Invalid Kernel Type while tryng to boot the NC. A: You didn't run the agree script. Q: I log in at the remote host and my keyboard settings fuck up. A: I use no .Xmodap when working on the NC. This keeps my keyboardsettings perfect.

6.2 Experiences

Using the NC as a plain terminal with multiple consoles. The Built-in Motiv extensions are a lot easier to use than the default setup ;-) Pressing the Pause/Break key will pop up the NC Menu at any time.
