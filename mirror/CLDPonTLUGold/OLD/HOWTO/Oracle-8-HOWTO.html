Oracle for Linux Installation HOWTO

Stephen Darlington, <stephen@zx81.org.uk>

$Revision: 1.14 $, $Date: 2000/06/19 21:07:43 $

With this HOWTO, and a little luck, you will be able to get "Oracle 8i Enterprise Edition for Linux" installed, create a database and connect to it from a remote machine. The main focus of this guide is RedHat Linux 6.0, although it should work more-or-less unchanged for any other recent distribution.
