15. More things to do

Here is a list of other interesting things to do with your Linux Intranet server. All the software mentioned below is freeware or shareware.

  • Browse the Linux server using Network Neighbourhood in Win95/ NT; Setup a WINS like NBT server. Check out the SAMBA Web page at http://lake.canberra.edu.au/pub/samba/samba.html
  • Implement a search engine on your Intranet. Connect to ht://Dig at http://htdig.sdsu.edu/
  • Use CUSeeMe by setting up a local reflector. Refer to their page at Cornell http://cu-seeme.cornell.edu/
  • Setup Web Conferencing. Use COW from http://thecity.sfsu.edu/COW/
  • Deploy a SQL database. Refer to the mSQL Home page at http://Hughes.com.au/
  • Setup FTP,Gopher,Finger,Bootp servers on the Netware server. Get them at http://mft.ucs.ed.ac.uk/
  • Emulate a Netware server. Check out the NCP Utilities at ftp://sunsite.unc.edu/pub/Linux/system/filesystems/ncpfs/
If you find other interesting things to do with your Linux Intranet server, please feel free to mail me.
