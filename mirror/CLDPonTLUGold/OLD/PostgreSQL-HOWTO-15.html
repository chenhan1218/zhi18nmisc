15. PostgreSQL 用的中央處理器

參閱文件 http://metalab.unc.edu/LDP/HOWTO/CPU-Design-HOWTO.html 以得到可用於 PostgreSQL 的中央處理器名單，本文也會提供有關中央處理器的詳情。

以下中央處理器 (CPUs)（32 位元和 64 位元）可執行 PostgreSQL。它們都運行 Linux。

  • 處理器主網站是︰Google 搜尋引擎 CPU 站“Computers>Hardware>Components>Microprocessors” http://directory.google.com/Top/Computers/Hardware/Components/Microprocessors

以下為 GNU/GPL 開放原始碼處理器名單︰

  • 開放原始碼處理器網站—Google 搜尋 "Computers>Hardware>Open Source" http://directory.google.com/Top/Computers/Hardware/Open_Source
  • OpenRISC 1000 Free 32-bit 處理器 IP 核心，與專斷的 ARM 和 MIPS 競爭，位於 http://www.opencores.org/cores/or1k
  • OpenRISC 2000 位於 http://www.opencores.org
  • 歐洲太空總署的 ESA-32bit 和 ESA-64bit 處理器“LEON”Sparc http://www.estec.esa.nl/wsmwww/leon
  • GNU/GPL Freedom 64-bit F-CPU http://www.f-cpu.org 或 http://f-cpu.tux.org 映射站台位於 http://www.f-cpu.de
  • STM 32 位元，雙向超純量 (2-way superscalar) RISC 處理器 http://www.asahi-net.or.jp/~uf8e-itu
  • 以 Verilog 或 VRML 寫成的免費微處理器和 DSP IP 核心 http://www.cmosexod.com
  • 可使開發加快的免費硬件核心 http://www.scrap.de/html/opencore.htm
  • Opencores org—開放原始碼、自由 IP core http://www.opencores.org
  • Linux 開放硬件和免費 EDA 系統 http://opencollector.org
  • ARM 處理器 http://www.arm.com/Documentation
  • Cogent 處理器 http://www.cogcomp.com
