11. PostgreSQL 的前端 GUI 工具（圖像使用者界面）

網頁瀏覽器在未來將會成為最流行的前端 GUI。建議你把所有「祖傳」的視窗 95/NT 程式改寫為在網上執行的程式。

你要使用網上應用程式伺服器如 Enhydra （使用 Java）或 Zope （使用 Python）或 OpenACS 。

最佳網上命令稿 (script) 編寫（或編譯）語言是 PHP+Zend 編譯器。PHP 功能十分強勁，因為它把 Perl、Java、C++ 和 Javascript 的功能結合到一種語言中，而且可於所有作業系統—Unixes 和視窗 NT/95 執行。

排名以喜歡的程度為先後，最佳工具為

  • Enhydra 在 Enhydra 加上 Borland Java JBuilder Linux 版 http://www.inprise.com
  • Zope 在 Zope
  • OpenACS 在 OpenACS
  • PHP 命令稿和 Zend 編譯器 PHP+Zend 編譯器
  • X-Designer 支援 C++, Java 和 MFC http://www.ist.co.uk/xd
  • Windows95 和 Unix 版的 QT 在 http://www.troll.no 和 ftp://ftp.troll.no
  • Code Crusader 在 Linux cdrom 中，根據 MetroWorks Code Warrior 寫成的免費軟件 http://www.kaze.stetson.edu/cdevel/code_crusader/about.html
  • MetroWorks 的 Code Warrior http://www.metrowerks.com
  • GNU Prof C++ IDE，來自 (Redhat) Cygnus http://www.cygnus.com
  • Borland C++ Builder Linux 版 http://www.inprise.com
  • Borland Java JBuilder Linux 版 http://www.inprise.com

以最佳為先的語言選擇—

  1. Java，但它的程式執行得很慢，而且要授權費。C++ 比 Java 快五倍！！
  2. Python（強大的物件導向命令稿語言）。
  3. PHP 萬維網伺服器命令稿加上 HTML、DHTML、Javascript 客戶端命令稿及 Java-Applet。
  4. Perl 命令稿語言使用 Perl-Qt 或 Perl-Tk Perl 資料庫介面
  5. Omnipresent 和 Omnipotent 語言 C++ (GNU g++)︰
    • Fast CGI（以 C++ 寫成）加上 Javascript/Java-Applet 作為網頁前端介面
    • GNU C++ 加 QtEZ 或 QT
    • GNU C++ 加 Lesstif 或 Motif。

還有其他工具—PostgreSQL 發行版本 (distribution) 中有一個名為‘pgTcl’的 Tcl/Tk 介面程式庫。Tcl/Tk 有一個名為 SpecTcl 的綜合開發環境。

  • Lesstif Motif 工具 ftp://ftp.redhat.com/pub/contrib/i386/lesstif*.rpm
  • Vibe Java/C++ 在 http://www.LinuxMall.com/products/00487.html
  • JccWarrior ftp://ftp.redhat.com/pub/contrib/i386/jcc*.rpm
  • Tcl/Tk http://www.scriptics.com
  • 名為 INCR 的 Tcl 物件導向延伸 http://www.tcltk.com
  • Visual TCL 網站 http://www.neuron.com
  • Visual TCL Redhat rpm 在 ftp://ftp.redhat.com/pub/contrib/i386/visualtcl*.rpm
  • http://sunscript.sun.com/
  • http://sunscript.sun.com/TclTkCore/
  • ftp://ftp.sunlabs.com/pub/tcl/tcl8.0a2.tar.Z
  • Java FreeBuilder ftp://ftp.redhat.com/pub/contrib/i386/free*.rpm
  • SpecTCL ftp://ftp.redhat.com/pub/contrib/i386/spec*.rpm
  • PostgreSQL 的 Java RAD 工具 Kanchenjunga http://www.man.ac.uk/~whaley/kj/kanch.html
  • Applixware 工具 http://www.redhat.com （譯按︰Redhat 早已停售 Applix 產品。）
  • XWPE X Window Programming Environment http://www.identicalsoftware.com/xwpe/ 或 ftp://ftp.rpi.edu/~payned/xwpe ftp://ftp.redhat.com/pub/contrib/i386/xwpe*.rpm
  • XWB X Window Work Bench ftp://ftp.redhat.com/pub/contrib/i386/xwb*.rpm
  • NEdit ftp://ftp.redhat.com/pub/contrib/i386/nedit*.rpm

你也可在視窗 95 中使用 Borland C++ Builder、Delphi、Borland JBuilder、 PowerBuilder 經 ODBC/JDBC 驅動程式連接到 unix 電腦的 PostgreSQL。
