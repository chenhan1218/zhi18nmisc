1. 關於這份文件

這是 Linux 高可用性 HOWTO 。 它預定成為發展 Linux 高可用性的白皮書。 （ Linux-HA 作為最初的標題）。 有興趣參與這項計畫的開發人員將在此找到在 Linux-HA 之中的基本概念與想法。 在發展過程中，這份文件將被部份的重寫以反映概念與實作的進度。 最終，這份HOWTO 將如其名所示成為： Linux-HA 軟體套件的手冊。

這份 HOWTO 是 Linux 文件計畫的一部分。 如果你需要知道 更多關於 Linux 文件計畫或關於整體的 Linux HOWTO ，儘管聯絡管理者 Greg Hankins 或參閱 HOWTO Index 。

Linux 高可用性 HOWTO 可以在以下網址找到 http://sunsite.unc.edu/pub/Linux/ALPHA/linux-ha/High-Availability-HOWTO.html 。 新版的 Linux 高可用性 HOWTO 總是最先被放置在那裡， 因此請先檢查一下你所正在閱讀的副本是最新的 ! 為了讓這更容易，有 變更紀錄 一節。

Linux-HA 也有個封閉的通信論壇。 "封閉的" 的意思是只有訂閱者可以張貼訊息到這個通信論壇以避免 spammer 寄給他們無用之物。 如果你有興趣，寄一封（空的）信件到 linux-ha-subscribe@muc.de 。 一旦你收到論壇管理者送出回的回覆，你必須回信加以確認。 這確保沒有人可以以他人的地址訂閱。 通信論壇本身是 linux-ha@muc.de 。
