CLDP 通信論壇

為了進一步整合翻譯工作，分享並傳承翻譯經驗，以及解決一些困難問題，本計畫設立了通信論壇(mailing list)。 地址是: ldp-trans@linux.org.tw。

本論壇的主要目的是:

  • 結合眾人的力量，整合 Linux HOWTOs 翻譯工作。
  • 討論 SGMLtools 中文處理問題。
  • 分享及傳承 Linux HOWTOs 翻譯的經驗。
  • 推廣 Linux HOWTOs 翻譯計畫。
  • 其它和 Linux 中文化相關的工作。

你還可在此找到信件收集。在通信論壇上提出問題前請先查閱一下舊信件上的討論。

注意: 請勿寄任何與上述主題無關的信件到通信論壇中，尤其是一些垃圾廣告信! 這將會引起眾人的憤怒...

使用方法:

  • 如果你想訂閱本論壇，請以 subscribe 為信件內容(非標題)寄到 ldp-trans-request@linux.org.tw 即可。
    注意這個地址，不要寄錯了! 如果訂閱成功了，會收到系統寄給你的歡迎信件。
  • 如果你想得到此通信論壇進一步說明，請以 info 為信件內容寄到上述地址。
  • 若想得到通信論壇的使用說明，請以 help 為信件內容寄到上述同一地址。
  • 如果你對通信論壇有任何建議或疑問，請寫信到 ldp-trans-approval@linux.org.tw。
  • 想取消訂閱的話，請以 unsubscribe 為內容寫信到 ldp-trans-request@linux.org.tw.

如果您願意協助翻譯工作，請加入此通信論壇。
