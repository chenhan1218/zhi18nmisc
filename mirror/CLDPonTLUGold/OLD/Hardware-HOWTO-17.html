17. 可移式磁碟機 (抽取式磁碟機)

只要控制卡有被支援，所有的 SCSI 磁碟機應該都可以使用，包含 光磁碟片 (MO), WORM, floptical, Bernoulli, Zip, Jaz, SyQuest, PD 等等的。

  • Panasonic MO (合併了 CD-ROM 磁碟機和可移式光學磁片的功能)。在建構 kernel 時需要打開某個開關才能同時使用這兩個功能。
  • 並列埠 Zip 磁碟機
    ftp://gear.torque.net/pub/
  • 並列埠 Avatar Shark-250
    http://www.torque.net/shark.html

抽取式磁片就像硬碟和軟碟片般地來使用，只需 fdisk/mkfs 和 mount 磁碟機即可。 Linux 提供了磁碟機鎖定功能，只要你的碟磁機有支援的話。如果磁片是 MS-DOS 格式的話，那麼mtools 也可以被使用。

CD-R 磁碟機需要特殊的軟體才能使用，請參閱 CD-R Mini-HOWTO。

Linux 有支援 512 和 1024 bytes/sector 的磁片。從 kernel 2.1.32 開始 Linux 也支援了 2048 bytes/sector的了。給 kernel 2.0.30 用的修補檔可以在下面取得。 http://liniere.gen.u-tokyo.ac.jp/2048.html

如果使用

  • Fujitsu magneto-optical disk drives M2513
的話，你會需要 2048 bytes/sector 的支援才可。

從 pre-patch-2.0.31-3 開始，IDE/ATAPI 內接式 Zip 磁碟機， flopticals 及 PD 都已被支援了。

  • LS-120 floptical
  • PD-CD
