12. PostgreSQL 的介面驅動程式

12.1 PostgreSQL 的 ODBC 驅動程式

ODBC 代表由微軟制定的‘Open DataBase Connectivity’。 它是從不同廠商的不同資料庫存取資料的流行介面。使用 ODBC 驅動程式寫成的軟件保證可用於不同資料庫，如 PostgreSQL、Oracle、Sybase 和 Informix 等。

  • PostODBC 已經包括在發行版本中。請參閱主網頁 http://www.postgresql.org。 它已包括在 PostgreSQL 光碟中。
  • 在 http://www.iodbc.org 的開放原始碼 ODBC 專案。
  • http://www.openlinksw.com Open Link Software Corporation 售賣 PostgreSQL 和其他資料庫的 ODBC 驅動程式。她也送出免費的 ODBC (名額有限）。
  • Insight 的 PostgreSQL ODBC http://www.insightdist.com/psqlodbc 這是 PostODBC 的官方網站。
  • FreeODBC 套件 http://www.ids.net/~bjepson/freeODBC/。這是一個免費的 ODBC。
  • PostgreSQL 的 ODBC 32 Explorer http://members.nbci.com/anhr

12.2 PostgreSQL 的 UDBC 驅動程式

UDBC 是一種獨立於驅動程式管理員 (driver managers) 和 DLL 支援的靜態 ODBC 版本，用來直接把資料庫連接能力嵌入到應用軟件中。

  • http://www.openlinksw.com Open Link Software Corporation 出售 PostgreSQL 和其他資料庫的 UDBC 驅動程式。Open Link 也送出免費的 UDBC (名額有限）。

12.3 PostgreSQL 的 JDBC 驅動程式

JDBC 代表‘Java DataBase Connectivity’。Java 是一種昇陽 (Sun Microsystems) 所開發的獨立於平台的程式語言。Java 程式設計師被鼓勵使用 JDBC 來撰寫資料庫程式，以增強程式在不同資料庫，如 PostgreSQL、Oracle、Informix 等之間的移植能力。如果你寫 Java 程式，你可在以下網站取得 PostgreSQL 的 JDBC 驅動程式。

JDBC 驅動程式已包括在 PostgreSQL 發行版本的 postgresql-jdbc*.rpm 套件中。

  • http://www.demon.co.uk/finder/postgres/index.html 昇陽 Java 的 PostgreSQL 連接
  • ftp://ftp.ai.mit.edu/people/rst/rst-jdbc.tar.gz
  • http://www.openlinksw.com Open Link Software Corporation 出售 PostgreSQL 和其他資料庫的 JDBC 驅動程式。Open Link 也送出免費的 JDBC (名額有限）。
  • JDBC 英國網站 http://www.retep.org.uk/postgres
  • JDBC FAQ 網站 http://eagle.eku.edu/tools/jdbc/faq.html

JDBC 的網頁、導引和 FAQ 在︰

  • JDBC HOME http://splash.javasoft.com/jdbc
  • JDBC 導引 http://www.javasoft.com/products/jdk/1.1/docs/guide/jdbc
  • JDBC FAQ http://javanese.yoyoweb.com/JDBC/FAQ.txt
請參考 測試 Java 介面 一節。
