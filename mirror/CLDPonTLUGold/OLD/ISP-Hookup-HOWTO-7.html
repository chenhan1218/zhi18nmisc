7. 後記

7.1 我應該知道的其他事情 ?

  • 正常上，系統埵釩雃h錯誤訊息都會被以內部電子郵件的方式來傳達。 想確認它們都會被正確地讀取，我們可以建立一個 /etc/aliases檔案。每一次你更動它時，記得用 newaliases 這個命令。 下面的例子幾乎包含了所有事項了：
PostMaster: root
ftp: root
news: root
usenet: root
FaxMaster: root
fax: root
WebMaster: root
MAILER.DAEMON: root

  • 很多的 Linux 版本的程式可在 Sunsite 上找到，這個站通常都很忙碌。 但有很多的映射站台 mirrors 它們會有相對應於 ftp://sunsite.unc.edu/pub/Linux/..的檔案， 所以你可以好好利用那些靠你家較近的映射站，例如 ftp://ftp.nvg.unit.no/pub/linux/sunsite/..。
  • 如果你高興從 Yarn 移植整個成為一個目錄，你也可以使用 yarn2mf來轉換它們成為標準的目錄，該程式可在下面站台取得

ftp://ftp.sol.no/user/egilk/yarn2mf.zip
