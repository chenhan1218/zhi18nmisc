29. 詞彙/術語

AGP

Accelerated Graphics Port 加速繪圖埠。是個匯流排互連機制，設計來改善 3D 三度空間繪圖應用。 AGP 是從繪圖子系統到內部邏輯晶片組間的專用匯流排 http://www.euro.dell.com/intl/euro/r+d/r+dnews/vectors/vect_2-1/v2-1_agp.htm

ATAPI

AT Attachment Packet Interface. 是新的協定用來控制大量儲存設備，和 SCSI 協定很像。他是在 ATA (AT Attachment) 介面上所建立的，正式的 ANSI 標準名稱為- 發展給硬碟使用的 IDE 介面。 ATAPI 通常運用在硬碟、 CD-ROM 磁碟機、磁帶機、以及其他設備上。

ATM

Asynchronous Transfer Mode 不同期傳輸模式

CDDA

?? CD-ROM/Writer 讀取音軌的能力。

DMA

Direct Memory Access 直接記憶體存取

EGA

Enhanced Graphics Adapter 加強型繪圖卡

EIDE

Enhanced IDE 加強型 IDE

EISA

Extended Industry System Architecture 擴充的工業系統架構

FDDI

Fiber Distributed Data Interface. 光纖分送資料介面，是高速環型區域網路。

IDE

Integrated Drive Electronics. 每一個磁碟機都有個內建的控制器。

ISA

Industry System Architecture 工業系統架構

ISDN

Integrated Services Digital Network 整合服務數位網路

MCA

MicroChannel Architecture 微通道架構

MFM

Modified Frequency Modulation 被修改的調變頻率 ?

MMX

Multimedia Extensions 多媒體延伸指令集。新加到Intel Pentium 處理器堶悸澈令集。它提供更佳的音效與影像品質。

PCI

Pheripheral Component Interconnect 週邊元件互連。由 Intel設計的32-位元匯流排。詳情可在 http://www.pcisig.com 和 http://infoserver.ee.siue.edu/~jbutter/EE580_1.html堶惕鋮魽C

RAID

Redudant Arrays of Inexpensive Disks 不貴的過剩磁碟機陣列。RAID 的基本概念來自於集結多個容量小、又不貴的磁碟機成為一個磁碟陣列。產生比一個容量大但是較貴的磁碟機還要好的效能。現在共有五種不同型式的過剩陣列架構；從 RAID-1 到 RAID-5。非過剩磁碟陣列也可稱為 RAID-0。
http://www.uni-mainz.de/~neuffer/scsi/what_is_raid.html

RLL

Run Length Limited 限制執行長度 ?

SCSI

Small Computer Systems Interface 小型電腦系統介面。定義來給電腦使用的設備小型架構。它可以讓你採用單一個轉接卡來連接全部的設備。
http://www.uni-mainz.de/~neuffer/scsi/what_is_scsi.html

SVGA

Super Video Graphics Adapter 超視訊繪圖轉接卡

UART

Universal Asynchronous Receiver Transmitter 通用異步接收/傳送器

VGA

Video Graphics Adapter 視訊繪圖轉接卡

VLB

VESA Local Bus

WORM

Write Once Read Many
