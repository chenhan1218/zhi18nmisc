6. 實際建議

最後，我想對想要成立和維持 LUG 並使發展壯大的同好提出一些非常實際甚至可算是俗不可耐的建議。

6.1 LUG 支助組織

以下是對區域性 LUG 提供協助的一些組織。

GLUE

(Groups of Linux Users Everywhere)是 SSC 成立的的一個項目，旨在協調和協助使用者 團體。他們是出版Linux Journal的同一批人。 GLUE 方案是區域性 LUG 可以用以向其成員提供一些協助的一條廉價途徑。

Linux Systems Labs

LSL 供應它們發行的 Linux 光碟片(四張光碟片上灌錄了三種 Linux 發 行版本：Red Hat、Slackware和Debian)給各地 LUG 團體，以便低價轉售。

Linux Mall User Group Program

由 WorkGroup Solutions贊助， Linux Mall User Group Program對參加的使用者團體提供各種便利。 LUG團體也可免費參加 Linux Mall's Referral Program。

Cleveland Linux User's Group

擁有網際網路域名lug.net。 它們可在lug.net上提供你們LUG團體的網際網路域名：你們的 LUG 名稱或城市 名稱lug.net。進一步資料請查詢 LUG.NET 或發送電子郵件給 Jeff Garvas.

Red Hat Software's User Group Program

協助各地 LUG 團體發展和成長。 進一步資料請查詢 Red Hat Web site

6.2 成立 LUG 團體

  • 查明附近已經存在的各個 LUG 團體
  • 在comp.os.linux.announce和在適當的本地社區宣佈打算成立 Linux 使用者協會
  • 在本地任何電腦使用者可能聚集的地點例如書店、swap meets、cybercafes、大學、學院、公司機構、網際網路公司等，宣佈打算成立 Linux 使用者協會，
  • 找到本地對 Linux 友好并可能願意協助你組織 Linux 使用者協會的公司或機構
  • 編制通訊名冊，聯絡有興趣成立使用者協會的同好，或建立其他通訊方式
  • 要求關鍵人物幫助你傳佈打算成立 Linux 使用者協會的消息
  • 設法在 Web 伺服器上編寫几頁 HTML 資料，說明想要成立的 LUG
  • 開始尋找開會場地
  • 安排舉行第一次會議的時間和地點
  • 在第一次會議中討論組成的 LUG 的工作目標

6.3 使用者協會的管理和成長

  • 儘量降低加入 LUG 成為會員的條件
  • 儘量優先利用 LUG 網址：隨時增訂網頁資料、 使網頁便于查找會議信息(會議的時間地點和討論主題)和突出聯絡人的資料和反饋方式
  • 幫助安裝 Linux 系統
  • 在本地電腦同好聚集的地點張貼佈告和散發傳單
  • 找到全心全意投入的負責人
  • 跟隨 Linus仁慈的獨裁的領導方式
  • 重大決定由會員投票決定
  • 編制提供技術支助的通訊名單，并要求“高手”加入
  • 組織高級和初級以及正式和非正式的討論會
  • 支持會員的軟件發展工作
  • 設法募集經費，而無需收集會費：例如出售Linux用品等方式
  • 考慮為協會取得合法地位，組成財團法人或免稅組織等
  • 了解會議地點是否限制協會人員的增加
  • 配合 swap meets、電腦產品展覽會或其他社區活動舉行會議。這種場合電腦使用者聚集，他們有可能改用Linux。
  • 按實際情況儘快選出使用者協會的正式領導人員：例如會長、財務、祕書、會議主持人、公關(處理Usenet和電子郵件事務及本地新聞)和會議協調員(組織使用者協會各次會議和安排演講人)
  • 提供途徑使會員能對使用者協會的方向、目標和行事策略提出反饋意見
  • 通過免費提供 Web 空間、通訊名單或 FTP 網址的方式，支持 Linux 和自由軟件的發展
  • 為有關軟件設立 FTP 網址
  • 儲存使用者協會在網址上的所有資料
  • 向 Linux 的供銷商和 VAR 等募集獎品供協會會議抽獎之用
  • 該給好處的就給好處
  • 加入SSC的GLUE (Groups of Linux Users Everywhere)，但需注意，這個組織征收會費
  • 將你們的使用者協會的資料送交全世界Linux使用者協會的名單
  • 在適當的 usenet 新聞組和本地電腦出版物和報刊發佈你們會議的消息
  • 印制宣傳材料，例如 Postscript 廣告，供會員在工作地點、書店或電腦商店張貼印發
  • 明確知道協會會員希望協會完成的工作
  • 向本地新聞渠道發送新聞，說明使用者協會的特別活動，例如免費安裝、網路學習等活動
  • 利用使用者協會的人力物力協助本地非盈利機構和學校所需的信息技術
  • 熱誠負責地推動 Linux 的使用
  • 運用使用者協會成員的才能
  • 維持與 Linux 銷售商、VAR 和開發商的良好關係
  • 查明本地 Linux 的技術顧問和保持聯系
  • 保持與本地或國內其他 Linux 使用者協會的聯系，以便交流經驗、訣竅和資源
  • 將 Linux 軟件的現況如新的內核、錯誤、修補和安全考慮以及新的連接埠、商標和許可証等問題隨時通知會員
  • 將使用者協會編制的文件，例如技術資料、教導材料和本地 HOWTO 等通知 Linux Documentation Project (LDP)和其他 Linux 有關的資料中心
