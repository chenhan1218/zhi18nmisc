2. 重要事項: 準備資訊及設定

2.1 對 Windows95 OSR2 (4.00.950b 版本) 使用者的警告事項:

千萬 不可 於未詳閱 Windows 95 OSR2 FAQ 的問題前試圖按 以 Windows 95 雙重開機. 該 FAQ 亦包含了許多對 FAT32 問題的探討. 大略來說, 如果你雖擁有 OSR2 版本但仍然使用 FAT16 檔案系統, 你仍可以使用 F4 功能鍵來作雙重開機.

注意: 如果你裝的是 Windows 95 OSR2 版本, 且使用 FAT32 檔案系統, 則本 mini-HOWTO 所述以 F4 功能鍵來作雙重開機的功能 無法使用而且切勿嘗試.

但是, 針對此, 我們有一些於開機時可自動執行 Linux 的替代方式, 將留待 第 6 章 討論.

2.2 設定前假定事項:

  • 你已成功的裝妥 Windows 95 系統於你的電腦.
  • Windows 95 是裝在電腦的 C 磁碟機.
  • 你已成功的裝妥 Linux 系統於你的電腦.
  • 你知道 Linux 系統裝在那一個分割區.
  • 你的硬碟中未安裝 LILO 程式.
  • 如果你使用 Windows 95 OSR2 (4.00.950b 版本), 你已詳讀 Windows95 OSR2 FAQ.

2.3 在開始前須準備的物品:

  • 一張具備 DOS 5.0 或備有 config.sys 組態檔的 DOS 6.x 系統檔案 (io.sys, msdos.sys, command.com) 的開機磁片. 如果你使用 DOS 5.0 開機磁片, 你還需要 autoexec.bat 開機執行檔. 如果你使用 Windows 95 OSR2 (4.00.950b 版本), 則你須要在 C 磁碟機的根目錄中以文書編輯軟體製作一個叫 Winboot.sys 的系統檔. Winboot.sys 檔案可以沒有任何資料, 只要你有這個檔案.

注意: 如果你的硬碟中已有 autoexec.dos, config.dos, io.dos, msdos.dos, 及 command.dos 等檔案, 則你不須要準備開機磁碟片. 你可以於 C 磁碟機的根目錄中鍵入: dir and dir /ah 來檢查是否有上述檔案. 此外, 如果你安裝的是 Windows 95 的升級版本, 則你應該已有該等檔案.

  • Loadlin.exe 程式檔案
  • 你的 kernel 映射檔案, 通常為 zImage 或 vmlinuz. (請參考 FAQ).
  • 簡單的文書編輯軟體, 像記事本 Notepad.
