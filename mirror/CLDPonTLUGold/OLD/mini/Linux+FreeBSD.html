這份文件途述如何在一個系統中使用 Linux 和 FreeBSD. 它介紹 FreeBSD, 並討論這兩個系統如何互相影響, 如一起分享 swap 空間. 你在讀這份文件之前, 應該要有使用 Linux 與做硬碟分割的經驗. 如果你對這篇文件有任何的評論, 問題, 與建議, 請不要猶豫, 寫封電子郵件給我. 我也希望聽聽有同時使用 Linux 和 NetBSD 或 OpenBSD 經驗的人說說.
