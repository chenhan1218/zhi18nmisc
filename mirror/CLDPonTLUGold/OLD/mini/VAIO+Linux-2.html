2. 硬體相容性

2.1 硬體和 Linux 不相容

在系統之中預先安裝的大多數硬體是和 Linux 相容，然而有一些例外，如數據機， USB 接頭和 MPEG 解碼器， 請注意你不能使用數據機，它會對你們其中一些人會有問題 ， 裝在 VAIO 之中的數據機是叫做 DSVD 數據機，它是 "Winmodem." 變型之一

(新加入: 12/28/98) 較新的型號好像是用標準的數據機代替 Winmodem。

2.2 部分相容的硬體

硬體中預先安裝的音效卡和顯示卡是部分相容的

音效卡

音效卡是使用微軟公司聲音系統 + 聲霸卡和 Linux 相容 ，如果你要在 Linux 之下使用 MIDI，你需要 調整你的聲音輸入能力，針對這種情況，取消 聲霸卡，如果你需要 MIDI，和聲音輸入，你可以考慮 買入 OSS/Linux，在 Linux 之下你可以使用它全部的功能， VAIO 的音效卡設備是 Yamaha OPL3SA， OSS/Linux 是 全部支援，它看起來像是 wavetable 規格中提到的綜合體 且是全部使用軟體達成，所以在這兒你不需要放棄硬體的能力 此時我寫到這兒，它的版本是 是 OSS/Free 3.8，它支援 Yamaha OPL3-SA1 audio controller 之下的音效卡，請注意 2.0.x 之中不是預先安裝的，你需要從 OSS 的網站下載， 2.1.x 跟著更新 OSS/Free 的版本而來，但是我沒有測試它的相容性 ，如果你發覺到它可以在音效卡上工作 ，請讓我知道。

顯示卡

在 Linux 之下的顯示卡是部分相容的，你 在 Linux console 模式之下使用是沒有問題，如果你希望在 X Window 之下使用， 要取得 3.3.1或者是之後的 XFree86 ，它支援 ATI Rage，別的 商業的 X Window 系統也支援，無論如何， 3D 顯示卡的 3D 加速功能在 Linux 之下沒有作用， 對於 3D 加速顯示卡 Linux 沒有驅動程式 ， (更新 12/29/98)對於型號 505， XFree86 V3.3.3 或者以後的 ALSA 驅動程式是相容的。
