7. User Groups

  • 參加當地的使用者協會 登記 Linux 使用者協會, 也是 Linux Documentation Project 的工作之一。 如果當地沒有 Linux 使用者協會, 就挺身組織一個。
  • Linux User Group HOWTO 這份文件當中談到許多發起使用者協會時要注意到的事情, 也談到使用者協會在推廣方面所扮演的重要角色。
  • 安排講者到對 Linux 有興趣的機構去演講。
  • 辦活動時, 記得向當地的新聞媒體發佈 新聞稿。
  • 主動為當地的組織機構設定 Linux 系統, 滿足使用者的需求。 當然也要包含對使用者的教育訓練及文件提供, 才能夠讓架起來的系統持續運作。
  • 在會議中討論這份 Linux Advocacy mini-HOWTO。 將各位集思廣益的結果告訴本文作者。
