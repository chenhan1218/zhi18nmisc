Linux 網路資源

與 Linux 有關的網站已不勝枚舉。這裡僅列出我們認為最重要，最有用的。 如果您發現值得推薦的站台，別忘了告訴我。

更新日期: 1999/12/26 15:15

中文站台:

BIG5 碼

Linux @ Taiwan
台灣 Linux 使用者組織。也是 CLDP 的主站台。
中文 Linux 延伸安裝套件
架構在 Red Hat Linux 上的中文延伸安裝套件。 終極目標在架構一個完全中文化的安裝套件 (distribution). 極力推薦。
LinuxFab
Linux 及時新聞。立志成為台灣的 Slashdot...:) 目前正對 Linux 的商業應用或研究計畫進行調查。如果您正在使用 Linux 可至該站登錄。
LinuxFocus 線上雜誌中譯版
LinuxFocus 電子雜誌中文翻譯。繼 CLDP 後另一個重要的 Linux 文件翻譯計劃。急需大家多多參與。
朝陽資管洪朝貴副教授的首頁
有許多介紹自由軟體，宣揚 GNU 理念的好文章。
XCIN Project
XCIN 專案計劃(X Window 的中文輸入法)的網頁。
Linux Notes
站長用 Linux 寫日記，記錄下測試過的各種好用軟體。還有站長自己寫的程式哦。很不錯的一個個人化網站。
GNU Translation Project - Chinese Team
一個將 GNU Tools 訊息中文化的翻譯計劃。需要大家多多參與。
Unix-like OS for PC
以介紹 Red Hat Linux 為主的站台。內容豐富，Red Hat 的使用者不可錯過。站長 marr.
Linux SAGA (原 OkStation)
站長 foxman 撰寫及翻譯了不少關於 Linux 的好文章。 包括很多 Linux 及 Windows NT 的比較。同時提供 FTP, CLDP 映射站等。是個相當用心經營的 Linux 站台。
微軟萬聖節備忘錄事件
微軟萬聖節備忘錄是由微軟內部所外洩的文件, 造成媒體及業界相當程度的震憾。 本文件對開放性原始碼的成功原因，以及對微軟帝國造成的威脅有深入的剖析。 OkStation 站長大手筆將此事件一系列相關文件中譯，值得大家鼓掌贊揚。
台大電機 Maxwell BBS 精華區
一個整理得相當完善的 Linux 精華區。
中研院 ASPAC 計畫文件
這是中研院內部的研究計畫，將多種公用軟體分類、測試後，整合。有許多好用工具的中文說明。
台灣 Linux 資源網
嘗試要從企業觀點、商業觀點的角度，做 Linux 相關資源的收集與推廣。 同時提供民調、雙向討論區等等互動服務，來收集台灣本地 Linux 使用族群的狀況。
探索 Linux
定期推出 Linux 應用的相關文章。同時有線上討論區。
交大資科 Linux 站
現有 FTP , World Wide Web, NFS-Install 等服務。 致力於本土 LINUX 作業系統的推廣與應用。 站上有不少中文文件，軟體套件等。並映射許多著名的 Linux 站台。
光碟月刊 Linux 文章索引
收集了曾經刊在光碟月刊(現改為 Internet Pioneer)上的 Linux 文章。
Clem's XEmacs Page
說明如何在 XEmacs 中設定使用中文。
Hanky's Linux Site
有 Linux Kernel 2.2.x 的測試報告，以及不少特殊的主題。
HKLUG Linux News
由香港 Linux 使用者協會所維護的 Linux 中文及時新聞。可惜目前計劃已停止。
Unix/Linux - voidspace
提供相當豐富的 Linux 中文資訊。站長就是上面提到的 Linux 中文新聞的維護者。
百資科技
代理各種 Linux 相關產品，及數十種程式語言工具。還有 Linux 的討論區。

GB 碼

北京 Linux 俱樂部
主要由北京市的大學生, 研究生，中科院、中關村等許多 Linux 的愛好者組成。 提供 Linux Today 中文版每日新聞。
南京 Linux 用戶俱樂部
有不少大陸上的 Linux 資源連結。
上海 Linux 用戶協會
上海 Linux 用戶協會網頁。
武漢自由軟件協會
武漢自由軟件協會網頁。
重慶 Linux 俱樂部
重慶 Linux 俱樂部網頁。
Chinese Linux User Group
Chinese Linux User Group 網頁。
中國軟件行業協會
國際自由軟件應用研究發展分會網頁。
Linux 公報中文版
Linux Gazette 的中譯版本。由南京 Linux 俱樂部所支持。 是大陸方面主要的中文 Linux 計劃。
中文 Linux 計劃
收集了不少 Linux 的中文軟件(GB 碼)，文章。
The Linux Homepage by Homer
介紹 Linux 的站台，有不少 FAQ, HOWTO 的收集。

英文站台:

FRESHMEAT
一個以介紹 Unix, Open Source 軟體為主的站台，每日更新。想知道軟體的最新消息不可錯過此站台。 上面還有 Linux 軟體的資料庫，方便大家尋找需要的軟體。 誰說 Linux 上的軟體不多? 看了這個站就知道 Open Source Community 的生產力驚人!
Slashdot: News for Nerds
Linux, Open Source 的及時新聞網，更新十分迅速。還有線上討論區，成為網上最重要的 Geek/Nerd 聚腳地，有不少著名的 Hacker 都是其常客。
Open Source: the Future is Here
由 Eric S. Raymond 所主持，開放性原始碼的總站台。有許多關於開放性原始碼的文獻資料。
Linux Weekly News
Linux 每週新聞。針對各主題分門另類介紹，包括核心，Distributions, 安全性，各平台移植等。每星期四更新。
LinuxFocus 線上雜誌
Linux 的電子雜誌。最值一提的是它有各語言的翻譯。中文版的翻譯計劃正熱列展開...:)
Linux Documentation Project
Linux 文件計劃。收集了 Linux FAQ, HOWTOs, Man pages, 以及各種文件指引等。這裡還收集了相當多的 Linux 連結及資訊。
LinuxHQ
介紹 Linux 核心為主的站台，包括新特徵的介紹，各版本的更新記錄，非正式的修補(patch)收集，以及更新核心的注意事項等。
Linux Standard Base
發展及提倡一個 Linux 的基本標準，以增加不同 Linux 發行套件間的相容性。 同時幫助軟體發展商移植並撰寫 Linux 的軟體。
Linux Reviews and Articles
蒐集了 Linux 文章以及各媒體上的 Linux 相關新聞。
