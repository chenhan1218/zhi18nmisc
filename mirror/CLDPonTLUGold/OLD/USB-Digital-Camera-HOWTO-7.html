7. 不要苦惱

更改SCRIPT FILE 可能會使你執行出錯,現在不理它們. 在你苦惱並抱怨它們沒有效果之前,看一下你的camera有沒有pictures. 如果你看過了, 請將 directory path names (路徑名稱)寫下來. 回到文檔編輯器中,將 dcim/100msdcf/*.jpg 替換之. 你可能需要用幾個不同的 script files 來處理不同的 picture 格式.

現在你的硬碟機的文檔目錄和 Section A.2 相似.執行並確認它的正確性:

bash $ dir -R name-of-your-picture-directory

希望這個文檔的資訊(有提到的manuals and HOWTOs)對你有用.
